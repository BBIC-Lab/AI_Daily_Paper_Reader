## 摘要
本研究针对具有明确输入输出关系的流式时间序列中的自适应建模问题。此问题的挑战在于：环境因素或输入时滞变化引起的系统快速变化（状态转变）会降低模型性能，且为每种时间序列模式使用多个小模型时，会面临准确性、鲁棒性与内存占用之间的权衡。为解决这些问题，本文提出了一种在线框架/方法，将流式时间序列视为时滞系统的动态混合。该框架通过使用固定长度的表征来概括历史状态，同时捕获系统动力学和输入输出时滞，从而保持模型跟踪的鲁棒性并减少内存占用。具体而言，该方法利用系统的马尔可夫参数序列构建摘要系统张量，同时捕获动态行为和时滞特性。必要时，张量分解算法可从张量中提取相关的历史模型，并帮助选择最适合当前状态的系统。该方法能够快速适应环境变化，且计算效率高。在真实数据集上的测试表明，DelayMix始终优于其他方法，尤其在高度非平稳数据上，实现了更优的预测精度和更快的时滞适应能力。

## Abstract
This research addresses the problem of adaptive modeling in time-series data streams with clear input-output relationships. This problem is challenging because rapid system changes (regime shifts) caused by environmental factors or input delay changes degrade model performance, and the trade-off among accuracy, robustness, and memory usage arises when using multiple small models for each time-series pattern. To address these issues, this paper presents an online framework/method that treats streaming time series as dynamic mixtures of time-delay systems. This framework maintains robustness of model tracking and reduces memory usage by summarizing past regimes using a fixed-length representation that captures both the system dynamics and input-output delays. Concretely, this approach constructs a summary system tensor using the system's Markov parameter series, capturing both dynamic behavior and delay characteristics. If necessary, a tensor decomposition algorithm extracts relevant past models from the tensor and helps select the system that best fits the current regime. This method enables rapid adaptation to environmental changes and is computationally efficient. Tests on real datasets show that DelayMix consistently outperforms other methods, achieving superior forecast accuracy and faster adaptation to delays, especially for highly non-stationary data.

## 1. 论文核心问题与研究动机

  - **非平稳性**：环境变化或输入延迟改变会导致系统状态（regime）发生**快速切换**，使事先训练好的模型迅速失效。
  - **资源约束**：流数据半无限长，若为每个模式维护独立模型，会陷入准确率、鲁棒性与内存/计算开销的三难困境。
- 现有流式预测方法未能同时处理**未知输入时滞**的跟踪问题，且模型数量随数据量单调增长，缺乏对时滞变化的自适应设计。
- 研究问题：能否开发一种方法，在**显式输入–输出关系的流式时间序列**上实现**高精度建模**，同时跟踪快速的环境变化并保持**高计算效率**？

## 2. 方法论：DelayMix 在线框架
### 2.1 核心思想
- 将流式时间序列建模为**多个未知时滞系统的动态混合**，每个状态（regime）对应一个时滞 MIMO 系统。
- 不直接估计时滞，而是通过**马尔可夫参数序列**（脉冲响应）来隐式捕获系统动态与延迟结构；对每个状态构造一个**等效的无时滞状态空间模型**用于推理与预测。
- 利用**固定长度的摘要系统张量**（基于高阶矩）总结历史状态，当检测到模型失配时通过**张量分解**提取各状态的马尔可夫参数，并按需更新模型库。

### 2.2 关键技术与算法流程
DelayMix 包含三个子算法：

1. **DynamicMomentCollection（动态矩收集）**  
   - 在滑动窗口上，利用当前数据 $X_c$ 和外生输入 $U_c$ 构建**三阶矩张量** $S$。  
   - 具体做法：对窗口内三元组 $(k_1, k_2, k_3)$，计算三个时刻的 $x(t) u(t)^\top$ 向量化外积，累积到 $S$ 的对应区块，实现**增量更新**，无需存储原始数据。

2. **ModelDatabaseAdaption（模型数据库适配）**  
   - 仅当当前模型 $\theta_p$ 对窗口数据的预测误差 $L_c$ 超过阈值 $\rho$ 时才触发。  
   - 采用 **ALS（交替最小二乘）算法**对系统张量 $S$ 进行秩-$R$ 的 CP 分解：
     $$ S \approx \sum_{i=1}^R q_i^{(1)} \otimes q_i^{(2)} \otimes q_i^{(3)} $$
   - 从分解因子重构各状态的**马尔可夫参数序列** $\hat{g}^{(i)}_k$，再利用 **Ho–Kalman 算法**实现为无时滞状态空间模型 $\{A_i, B_i, C_i\}$（定理 1 保证输入–输出等效）。  
   - 选择在历史窗口上误差最小的系统作为当前系统 $\theta$。

3. **FuturePrediction（未来预测）**  
   - 基于选定的无时滞模型，通过 Kalman 滤波/平滑估计当前隐状态初值，再代入未来外生输入 $U[t_n+1:t_n+l_s]$ 进行**多步外推预测**。

**理论支撑**  
- 定理 1：对有最小时滞的系统，任意极小无时滞实现若马尔可夫参数匹配，则输入–输出等价。  
- 定理 2（引用 Bakshi et al., 2023）：在适当假设下，系统张量的 CP 分解可恢复各状态马尔可夫参数（至置换和缩放）。  
- 计算复杂度与数据流长度无关，主要开销来自 ALS 分解，但因仅在模型失配时运行且使用热启动，实际耗时很低。

## 7. 优点
- **创新性强**：首次将高阶矩张量分解引入流式时滞系统混合建模，理论支撑扎实（引理、定理完备）。
