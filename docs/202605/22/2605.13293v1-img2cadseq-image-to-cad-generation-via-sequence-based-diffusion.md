## 摘要
边界表示（BRep）是计算机辅助设计（CAD）的标准格式，然而由于拓扑约束和操作序列的复杂性，从单视图图像重建高质量BRep仍然具有挑战性。我们提出了Img2CADSeq，这是一个多阶段流水线，通过将CAD序列编码为三级层次码本来克服这些限制。在重要性优先策略的指导下，该策略重视轮廓而非细节，将长序列压缩到稳定的离散潜在空间中。为了弥合模态差距，我们利用从粗到细的点云中间表示，通过对比学习将2D视觉特征与3D CAD序列对齐，以条件化VQ-Diffusion模型。在新引入的CAD-220K和PrintCAD数据集的支持下，我们的方法确保了稳健的工业领域适应。大量实验表明，Img2CADSeq显著优于最先进的方法，生成的标准STEP文件可直接用于商业CAD软件。

## Abstract
Boundary Representation (BRep) is the standard format for Computer-Aided Design (CAD), yet reconstructing high-quality BReps from single-view images remains challenging due to the complexity of topological constraints and operation sequences. We present Img2CADSeq, a multi-stage pipeline that overcomes these limitations by encoding CAD sequences into a three-level hierarchical codebook. Guided by an importance prioritization, this strategy values profiles over details, compressing long sequences into a stable discrete latent space. To bridge the modality gap, we leverage a coarse-to-fine point cloud intermediate, aligning 2D visual features with 3D CAD sequences via contrastive learning to condition a VQ-Diffusion model. Supported by newly introduced CAD-220K and PrintCAD datasets, our approach ensures robust industrial domain adaptation. Extensive experiments demonstrate that Img2CADSeq significantly outperforms state-of-the-art methods, producing standard STEP files that can be directly used in commercial CAD software.

## 2. 方法论

Img2CADSeq 的整体框架包含三个关键阶段：

### 2.1 层次序列编码与三级码本

- **层次结构**：将 CAD 操作序列分解为三级层次，由粗糙到精细：
  - **Extrude-Block (EB)**：编码全局语义（草图平面法向、原点、拉伸深度、布尔操作类型）。
  - **Sketch-Patch (SP)**：通过面积、对角线等得分排序捕获轮廓的拓扑布局，为下层提供空间锚点。
  - **Curve-Cluster (CC)**：采用局部 Frenet 坐标（弦长 $l_i$、相对角度偏差 $\Delta\theta_i$、曲率 $\kappa_i$、残差偏移）编码几何曲线，避免全局坐标依赖。
- **量化与训练**：每一层独立使用 VQ-VAE 将连续参数映射为离散码本，训练损失包括重建损失（MSE + 交叉熵）、量化损失和闭环正则项 $L = L_{\text{recon}} + L_{\text{vq}} + \lambda_{\text{cls}} L_{\text{closure}}$。

### 2.2 几何感知的特征提取

- **点云提升**：基于 Dens3R 将单视图图像提升为粗点云，仅微调 Query 和 Value 层（PEFT）以适应工业几何特性。
- **UA‑DGCNN 与启发式重采样**：预测每个点的几何重要性分数 $s_i \in [0,1]$，再通过混合分布（显著性项 + 均匀覆盖项）进行重采样，以保留锐利边缘和平滑表面，得到精细点云及其嵌入 $z_P$。

### 2.3 跨模态对齐与生成

- **对比学习对齐**：将点云嵌入 $z_P$ 与 CAD 序列的因果 Transformer 全局嵌入 $z_S$ 用 InfoNCE 损失对齐，建立共享流形。
- **条件扩散生成**：将对齐后的条件 $c = z_P$ 注入 VQ‑Diffusion 模型，通过逆扩散过程恢复 CAD 离散 token，再利用几何内核编译为 BRep 模型。
- **训练目标**：扩散模型结合变分下界和辅助交叉熵损失 $L_{\text{gen}} = L_{\text{vlb}} + \lambda_{\text{aux}} \mathbb{E}_{t,x_t}[-\log p_\theta(x_0|x_t,c)]$。

## 8. 不足与局限

- **单视图模糊性**：在严重遮挡区域，模型可能“幻觉”出看似合理但物理上不可制造的后部结构。
