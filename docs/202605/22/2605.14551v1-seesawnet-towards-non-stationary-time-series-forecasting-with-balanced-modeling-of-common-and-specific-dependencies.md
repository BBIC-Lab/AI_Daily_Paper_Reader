## 摘要
实例归一化（IN）广泛应用于非平稳多变量时间序列预测，以减少分布偏移并突出样本间的共同模式。然而，IN 可能过度平滑实例特定的结构信息，而这对于建模时序和跨通道异质性至关重要。以往方法进一步抑制分布差异或试图恢复时序特定依赖，但常常忽略一个核心矛盾：如何根据每个实例的非平稳结构自适应地建模共同与实例特定的依赖。为解决这一困境，我们提出 SeesawNet，一个在时序和通道维度上动态平衡共同与实例特定依赖建模的统一架构。其核心是自适应平稳-非平稳注意力（ASNA），它从归一化序列中捕获共同依赖，从原始序列中捕获特定依赖，并根据实例级非平稳性自适应融合两者。基于 ASNA，SeesawNet 交替进行专用的时序和通道关系建模，以联合捕获长程和跨变量依赖。在多个真实世界基准上的大量实验表明，SeesawNet 始终优于最先进的方法。

## Abstract
Instance normalization (IN) is widely used in non-stationary multivariate time series forecasting to reduce distribution shifts and highlight common patterns across samples. However, IN can over-smooth instance-specific structural information that is essential for modeling temporal and cross-channel heterogeneity. While prior methods further suppress distribution discrepancies or attempt to recover temporal specific dependencies, they often ignore a central tension: how to adaptively model common and instance-specific dependency based on each instance's non-stationary structures. To address this dilemma, we propose SeesawNet, a unified architecture that dynamically balances common and instance-specific dependency modeling in both temporal and channel dimensions. At its core is Adaptive Stationary-Nonstationary Attention (ASNA), which captures common dependencies from normalized sequences and specific dependencies from raw sequences, and adaptively fuses them according to instance-level non-stationarity. Built upon ASNA, SeesawNet alternates dedicated temporal and channel relationship modeling to jointly capture long-range and cross-variable dependencies. Extensive experiments on multiple real-world benchmarks demonstrate that SeesawNet consistently outperforms state-of-the-art methods.

## 2. 方法论：核心思想、关键技术细节与算法流程
- **总体思想**：SeesawNet 是一种在时间和通道两个维度上动态平衡“共同依赖”与“实例特定依赖”的统一架构。其核心是可复用的 **自适应平稳-非平稳注意力模块 (ASNA)**。
- **输入预处理 (Patching & Embedding)**：
  - 对输入 $X\in \mathbb{R}^{C\times L}$，分别计算平稳序列 $X_{\text{sta}} = \text{IN-Norm}(X)$ 和原始非平稳序列 $X_{\text{non}} = X$。
  - 两者各自分块（overlapping patches，长度 $P$，步长 $S$），经嵌入后得到 $P_{\text{sta}}, P_{\text{non}} \in \mathbb{R}^{C\times N \times D}$，其中 $N$ 为 patch 数量。
- **ASNA 模块**（图 3(c)）：
  - **平稳注意力**：对归一化后的 token 序列 $Z_{\text{sta}}$ 计算注意力 $A_{\text{sta}} = \text{softmax}(Z_{\text{sta}}Z_{\text{sta}}^\top/\sqrt{d})$，捕捉可迁移的共性模式。
  - **非平稳注意力**：用原始 token $Z_{\text{non}}$ 计算注意力 $A_{\text{non}} = \text{softmax}(Z_{\text{non}}Z_{\text{non}}^\top/\sqrt{d})$，突出实例特定的结构信息。
  - **自适应门控**：$G = \sigma(\text{Linear}(Z_{\text{sta}}\oplus Z_{\text{non}}))$，为每对 token 动态生成融合权重。
  - **融合输出**：$O = ((1-G)\odot A_{\text{sta}} + G\odot A_{\text{non}})Z_{\text{sta}}$，其中 value 统一采用 $Z_{\text{sta}}$ 以保持嵌入空间一致。
  - 采用残差连接和前馈网络更新 $Z_{\text{sta}}$ 和 $Z_{\text{non}}$：$\hat{Z}_{\text{sta}} = \text{Norm}(Z_{\text{sta}} + \text{Dropout}(O)) + \text{FFN}(\cdots)$，$\hat{Z}_{\text{non}}$ 类似。
- **Patch Dependency Learning Layer（时序依赖层）**：
  - 对每个通道 $c$，将其 patch 嵌入 $P_{\text{sta}}[c,:,:]$ 和 $P_{\text{non}}[c,:,:]$ 送入 ASNA（此时 $Z$ 为 patch token，长度 $T=N$），得到更新后的时序表示，从而在每个通道内平衡长程共性动态与实例特定变化。
- **Channel Relationship Learning Layer（通道依赖层）**：
  - 先用一维卷积压缩 patch 数量，得到 $Q_{\text{sta}}, Q_{\text{non}}\in \mathbb{R}^{C\times N'\times D}$。
  - 对每个时间步 $n$，将通道嵌入 $Q_{\text{sta}}[:,n,:]$ 和 $Q_{\text{non}}[:,n,:]$ 送入 ASNA（此时 $Z$ 为通道 token，长度 $T=C$），实现跨变量依赖的自适应平衡，使模型能捕捉不同时段下可能反转或变化的通道关系。
- **预测头与反归一化**：
  - 将更新后的 $\hat{Q}_{\text{sta}}$ 展平，经线性/MLP 预测头得到 $Y'$，再通过 $\text{IN-Denorm}$ 恢复原始尺度：$\hat{Y} = \text{IN-Denorm}(Y')$。
- **训练目标**：使用混合时频损失 Fredf（频率域抑制预测误差自相关）提升稳定性。

- SeesawNet 在 72 个设定中的 51 个取得最优，其余多为次优，显著优于 IN 改进型、后归一化依赖建模方法以及强 backbone。
