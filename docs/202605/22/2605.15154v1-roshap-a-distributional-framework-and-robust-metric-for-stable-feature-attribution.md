## 摘要
特征归因分析对于解释机器学习模型并支持可靠的数据驱动决策至关重要。然而，特征归因度量通常表现出随机变动：不同的训练-测试集划分、随机种子或模型拟合过程可能产生显著不同的归因值和特征排序。本文提出了一个框架，以纳入特征归因的随机性质，并基于SHAP度量提出一种稳健的归因度量RoSHAP，用于稳定特征排序。该框架对特征归因得分的分布进行建模，并通过自助重采样和核密度估计来估计。我们证明，在温和的正则性条件下，聚合特征归因得分是渐进高斯的，这大大降低了分布估计的计算成本。RoSHAP将SHAP的分布总结为一个稳健的特征排序准则，同时奖励那些活跃、强效且稳定的特征。通过模拟和真实数据实验，所提出的框架和RoSHAP在识别信号特征方面优于标准单次归因度量。此外，使用RoSHAP所选特征构建的模型，在预测性能上可与全特征模型相媲美，同时使用的预测变量却少得多。所提出的RoSHAP方法改善了机器学习模型的稳定性和可解释性，能够为分析提供可靠且一致的见解。

## Abstract
Feature attribution analysis is critical for interpreting machine learning models and supporting reliable data-driven decisions. However, feature attribution measures often exhibit stochastic variation: different train--test splits, random seeds, or model-fitting procedures can produce substantially different attribution values and feature rankings. This paper proposes a framework for incorporating stochastic nature of feature attribution and a robust attribution metric, RoSHAP, for stable feature ranking based on the SHAP metric. The proposed framework models the distribution of feature attribution scores and estimates it through bootstrap resampling and kernel density estimation. We show that, under mild regularity conditions, the aggregated feature attribution score is asymptotically Gaussian, which greatly reduces the computational cost of distribution estimation. The RoSHAP summarizes the distribution of SHAP into a robust feature-ranking criterion that simultaneously rewards features that are active, strong, and stable. Through simulations and real-data experiments, the proposed framework and RoSHAP outperform standard single-run attribution measures in identifying signal features. In addition, models built using RoSHAP-selected features achieve predictive performance comparable to full-feature models while using substantially fewer predictors. The proposed RoSHAP approach improves the stability and interpretability of machine learning models, enabling reliable and consistent insights for analysis.

### 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：特征归因度量本身具有**随机性**。由于训练数据划分、随机种子、模型初始化等不确定因素，同一特征在多次独立分析中可能得到差异悬殊的归因值，导致**特征重要性排序极不稳定**。这一现象在高维、小样本数据（如基因组学）中尤为突出。
- **整体含义**：本工作不再将特征归因视为一个固定值，而是**将其看作一个随机变量的实现**，通过估计归因得分的分布，构建一个能同时奖励“贡献频繁、幅度大且跨次运行稳定”的**鲁棒特征排序度量 RoSHAP**，从而提升机器学习解释的可靠性与一致性。

### 2. 论文提出的方法论

#### 2.1 核心思想
- 基于 SHAP 值，通过**自助法（bootstrap）** 对原始数据反复重采样、重新训练模型、重新计算 SHAP，获得特征归因的**经验分布**。
- 利用**核密度估计（KDE）** 或**渐进高斯近似**总结该分布，再从中提炼一个综合性的单值鲁棒指标。

#### 2.2 关键技术细节
- **样本级零膨胀建模**：单个样本‑特征 SHAP 值 $T_{ij}$ 建模为 $T_{ij} \sim w_{ij}\delta_0 + (1-w_{ij})\,G_{ij}$，其中 $\delta_0$ 表示零值点质量，$G_{ij}$ 为非零部分。
- **特征级聚合量**：定义 $U_j = \sum_{i=1}^{n} |T_{ij}|$，分解为 $|T_{ij}| = (1-B_{ij})H_{ij}$，$B_{ij}\sim \text{Bernoulli}(w_{ij})$ 为是否为零的指示，$H_{ij}=|G_{ij}|$。
- **渐进正态性**：在满足 $B_{ij}$ 与 $H_{ij}$ 独立，且 Lyapunov 条件成立时，定理给出：
  $$
  \frac{U_j - \mu_j}{s_j} \xrightarrow{d} \mathcal{N}(0,1),
  $$
  其中 $\mu_j = \sum_i (1-w_{ij})\mathbb{E}(H_{ij})$，
  $s_j^2 = \sum_i (1-w_{ij})\text{Var}(H_{ij}) + \sum_i w_{ij}(1-w_{ij})\{\mathbb{E}(H_{ij})\}^2$。
  该近似大幅减少了所需的 bootstrap 次数。
- **RoSHAP 指标**：对特征 $j$，定义  
  $P_{0j} = \Pr(U_j=0)$（零值概率），  
  $m_j = \text{Median}(U_j \mid U_j>0)$（非零中位数），  
  $s_j = \text{SD}(U_j)$（标准差）。  
  鲁棒特征归因得分为  
  $$
  \text{RoSHAP}_j = (1-P_{0j})\frac{m_j^2}{s_j},
  $$
  同时奖励**活性**（$1-P_{0j}$）、**强度**（$m_j$）和**稳定性**（$1/s_j$）。

#### 2.3 算法流程
1. 预处理数据，可选初步筛选预测变量。
2. 进行 $B$ 次 bootstrap 迭代：抽取自助训练集，保留袋外观测（OOB）作为计算 SHAP 的数据。
3. 每次迭代拟合模型 $\hat{f}^{(b)}$，计算 OOB 的 SHAP 矩阵 $T_{ij}^{(b)}$。
4. 聚合每轮特征级贡献 $U_j^{(b)}$。
5. 基于 $B$ 次实现的 $U_j$ 用 KDE（或高斯近似）估计分布，进而计算 $P_{0j}$、$m_j$、$s_j$ 及 RoSHAP 值。

### 6. 论文的主要结论与发现

- **分布视角提升稳定性**：将归因得分视为分布，能够有效刻画不确定性，避免因单次随机性而错误排名的信号特征。

- **高斯近似减少计算**：在满足 Lyapunov 条件时，$U_j$ 近似正态，使得可以用少量 bootstrap 重复（如 10 次）获得可靠的分布总结，计算成本显著降低。

### 7. 优点：方法或实验设计上的亮点

- **分布建模 + 零膨胀处理**：显式分离特征归因中的“零值比例”与“非零部分”，贴合树模型 SHAP 常出现精确零值的特点。
- **渐进理论支撑**：提供了特征级聚合量的渐近正态性证明，不仅节约计算，而且给统计推断留下空间。
- **统一的鲁棒指标**：RoSHAP 将活性、强度、稳定性三个维度整合为一个分数，可直接用于特征排序，实务上简洁易用。
- **跨任务与跨模型验证**：在表格数据、图像数据，以及树集成、线性模型上均展示了方法的有效性，通用性强。
