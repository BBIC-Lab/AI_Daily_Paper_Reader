## 摘要
多变量时间序列异常检测在实际应用中日益重要，但标注数据常常稀缺。许多现有方法依赖无监督学习来建模正常模式，然而它们通常平等对待所有通道。这种设计可能稀释与异常相关的信号，因为并非所有通道对异常检测的贡献相同。在本文中，我们提出 CALAD，一个面向多变量时间序列异常检测的通道感知对比学习框架。CALAD 利用估计的通道相关性来控制对比样本的构建，使学习过程反映异常语义而非通用相似性。通道相关性通过基于 Transformer 的自编码器的重构误差进行估计，并用于区分对异常行为影响更大的通道。利用这一信息，我们设计了一种通道级增强策略，其中正负样本根据是否保留或扰动异常相关通道来构建。这促使模型对无关通道的变化保持不变性，同时对异常相关通道的变化敏感。此外，CALAD 结合了对比学习和辅助重构头，使模型能够学习判别性表征同时保留正常结构。在多个真实世界数据集上的实验表明，CALAD 始终优于现有方法，尤其在分布偏移场景下。我们提供代码以促进可复现性：https://github.com/hirundo1218/CALAD。

## Abstract
Multivariate time series anomaly detection has become increasingly important in real-world applications, where labeled data are often scarce. Many existing approaches rely on unsupervised learning to model normal patterns, but they often treat all channels equally. This design can dilute anomaly-relevant signals, since not all channels contribute equally to anomaly detection. In this paper, we propose CALAD, a channel-aware contrastive learning framework for multivariate time series anomaly detection. CALAD governs the construction of contrastive samples using estimated channel relevance, allowing the learning process to reflect anomaly semantics rather than generic similarity. Channel relevance is estimated from reconstruction errors of a transformer-based autoencoder and is used to distinguish channels that are more influential to anomalous behaviors. Using this information, we design a channel-wise augmentation strategy in which positive and negative samples are constructed based on whether anomaly-relevant channels are preserved or perturbed. This encourages invariance to changes in irrelevant channels while being sensitive to changes in anomaly-relevant channels. Furthermore, CALAD combines contrastive learning and an auxiliary reconstruction head, allowing the model to learn discriminative representations while retaining normal structures. Experiments on multiple real-world datasets shows that CALAD consistently outperforms existing methods, particularly under distribution shift scenarios. We provide the code for reproducibility at https://github.com/hirundo1218/CALAD

## 2. 论文提出的方法论

### 2.1 核心思想
CALAD 不将通道重要性作为静态权重或注意力，而是直接利用估计的通道相关性来**决定对比样本的生成策略**：
- **正样本**：保留异常相关通道，仅扰动无关通道，迫使模型学习多样但正常的模式。
- **负样本**：扰动异常相关通道，保留无关通道，使产生的视图携带“异常”语义，模型应将其推远。

### 2.2 关键技术细节与流程

CALAD 由四个阶段组成：

#### (a) 通道相关性估计
- 先训练一个基于 Transformer 的自编码器对所有通道进行重构。
- 计算逐点重构误差 $e_{i,c} = |x_{i,c} - \hat{x}_{i,c}|$，并将窗口级平均重构误差作为响应变量 $y_i = \frac{1}{C}\sum_{c} e_{i,c}$。
- 以原始多变量输入为预测变量，利用 **LASSO 回归**估计系数：
  $$ \hat{\beta} = \arg\min_{\beta} \left( \|y - X\beta\|_2^2 + \lambda \|\beta\|_1 \right) $$
  非零系数对应的通道被标记为「异常相关」，零系数通道为「异常无关」。

#### (b) 通道感知的正负样本构建
- 根据通道标签 $label(c) \in \{0,1\}$，基于 **逆 FFT 扰动** 选择性地增强：
  - 正样本：对 $label(c)=0$ 的通道扰动，保留相关通道 → $w_i^{pos} = \text{Augment}_{\text{iFFT}}(w_i^{anc}, \{c \mid label(c)=0\})$。
  - 负样本：对 $label(c)=1$ 的通道扰动，保留无关通道 → $w_i^{neg} = \text{Augment}_{\text{iFFT}}(w_i^{anc}, \{c \mid label(c)=1\})$。
- 为每个原始窗口（anchor）构建三元组 $(w_i^{anc}, w_i^{pos}, w_i^{neg})$。

#### (c) 最近邻搜索
- 使用编码器 $f(\cdot)$ 提取三元组嵌入 $z_i^{anc}, z_i^{pos}, z_i^{neg}$，并用三元组损失训练该编码器：
$$ L_T = \frac{1}{N} \sum_{i=1}^N \max\left( \|z_i^{anc} - z_i^{pos}\|_2^2 - \|z_i^{anc} - z_i^{neg}\|_2^2 + \alpha, 0 \right) $$
- 训练后，对每个 anchor 在嵌入空间中检索其 **最近邻** 和 **最远邻**，形成新的三元组 $(w_i^{anc}, w_i^{near}, w_i^{fur})$。

#### (d) 结构保持的对比学习
- 将新三元组通过线性投影和 Transformer 编码器获得嵌入 $(z_i^{anc}, z_i^{near}, z_i^{fur})$。
- **对比损失**：基于 softmax 相似度的二值交叉熵，拉近 anchor 与近邻，推远 anchor 与远邻。
- **重构损失**：仅对 anchor 嵌入使用 MLP 重构原始窗口，$\hat{w}_i^{anc} = f_{mlp}(z_i^{anc})$，损失为 MSE。
- 联合优化：$L_{Total} = L_{contra} + L_{rec}$。

#### (e) 异常评分
- 模型输出 $K=10$ 个潜在类别的概率，训练集中出现频率最高的类为「正常类」。
- 测试时，若窗口属于正常类的概率 $p_{normal}(w_i)$ 最高则为正常，否则为异常，异常得分 $= 1 - p_{normal}(w_i)$。

## 6. 论文的主要结论与发现

- CALAD 在 **MSL、SMAP、SMD** 上取得最优 F1，在 SWaT 上以微小差距位列第二，平均相对提升 11%。
- **分布偏移优势明显**：尤其在 MSL 和 SMAP 上，CALAD 分别提升 31% 和 13% 的 F1，归功于通道感知增强保留了异常相关通道，使模型能学习到更多样的正常模式，从而更好地识别未见过的正常状态。
- 通道感知增强策略和逆 FFT 扰动显著优于全通道均匀增强或异常注入。
- 联合对比损失与重构损失可促使嵌入空间更清晰分离，并保留正常结构，有效降低误报。
- Transformer 编码器结合 MLP 重构头比纯 ResNet 或纯 Transformer 效果更佳。

## 7. 优点

- **创新性强**：首次将通道相关性估计直接融入对比样本的构造逻辑，而非仅作为权重或选取特征，实现了对比目标与异常语义的对齐。
- **完全无监督**：通道相关性通过重构误差和 LASSO 估计，无需任何异常标签。
- **结构保留机制**：在对比学习基础上引入辅助重构损失，约束 anchor 保留正常模式，防止表示学习偏离数据本真结构，设计巧妙。
- **评估严谨**：主动弃用 Point Adjustment，采用原始 F1 等指标，使比较更可信。
- **对抗分布偏移**：通过扰动无关通道生成多样正样本，有效提升了模型对未见正常模式的泛化能力。

## 8. 不足与局限

- **流程复杂性高**：方法包含四个串行阶段，需要训练额外自编码器、LASSO 估计、最近邻搜索网络以及最终模型，工程实现和调参成本较高。
- **通道估计误差风险**：LASSO 依赖重构误差的质量，若自编码器未能捕捉到某些异常通道的重构偏差，可能导致相关性估计错误，进而误导增强策略。
- **单一实体泛化**：在 SWaT 等实体数较少的场景下增益不明显，通道级语义多样性的优势未能充分发挥。
- **超参数敏感性未深入探讨**：LASSO 正则化系数 $\lambda$ 等关键参数仅提及经验设定，未分析其对性能的鲁棒性。
- **计算开销**：虽未明确报告，但多阶段训练和最近邻检索在数据量极大时可能带来较大开销，影响实时应用。
- **评估范围**：仅在四个常见基准上测试，缺乏更多样化的领域数据（如金融、医疗）验证。
