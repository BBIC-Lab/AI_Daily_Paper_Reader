## 摘要
在概率多变量时间序列（MTS）预测中，有效建模非平稳动态需要在表达能力和稳健性之间取得平衡。现有的参数方法得益于强大的归纳偏置但缺乏灵活性，而深度生成模型在没有大量数据和计算的情况下难以捕获复杂的时间依赖性。我们提出了参数先验映射（PPM），这是一个将参数结构先验注入到生成建模过程中的框架。具体来说，PPM利用参数估计器推导出一个动态、自适应的先验，该先验通过可学习的映射指导复杂预测分布的学习。这种设计使模型在保留参数方法效率的同时，充分利用生成模型的表达能力。通过混合目标进行训练，PPM能够产生精确的预测，并具有良好的校准不确定性估计。实证结果表明，PPM在处理非平稳数据方面优于现有基线，在准确性和计算效率之间提供了更优的权衡。代码可在https://github.com/ljl8336/PPM获取。

## Abstract
Effectively modeling non-stationary dynamics in probabilistic multivariate time series(MTS) forecasting requires balancing expressiveness with robustness. Existing parametric approaches benefit from strong inductive biases but lack flexibility, whereas deep generative models struggle to capture complex temporal dependencies without extensive data and computation. We introduce Parametric Prior Mapping (PPM), a framework that injects parametric structural priors into a generative modeling process. Specifically, PPM utilizes a parametric estimator to derive a dynamic, adaptive prior that guides the learning of a complex predictive distribution via a learnable mapping. This design allows the model to retain the efficiency of parametric methods while exploiting the expressive power of generative models. Trained with a hybrid objective, PPM yields precise forecasts with well-calibrated uncertainty estimates. Empirical results show that PPM outperforms existing baselines in handling non-stationary data, offering a superior trade-off between accuracy and computational efficiency. The code is available at https://github.com/ljl8336/PPM.

  - 现有扩散模型试图改进先验匹配（如 TMDM 用固定方差的高斯，NsDiff 用滑动窗估计方差），但仍难以灵活追踪时变的数据分布和随机性。

### 2. 方法论
- **核心思想**：利用参数估计器快速构造一个动态、自适应的条件先验，再通过一个可学习的映射将该先验“推前”到复杂的目标预测分布，从而在保留参数化效率的同时获得生成建模的表达能力。
- **关键技术细节与流程**（训练分三阶段）：
  1. **参数先验诱导**：
     - 编码器 $f_\theta(\cdot)$（可选用 MLP 等轻量架构）从历史窗口 $\mathbf{x} \in \mathbb{R}^{H \times C}$ 提取统计量：  
       $$[\boldsymbol{\mu}, \boldsymbol{\sigma}] = f_\theta(\mathbf{x}), \quad \boldsymbol{\mu}, \boldsymbol{\sigma} \in \mathbb{R}^{C \times D}$$
     - 定义条件先验分别为对角高斯：  
       $$p_\theta(\mathbf{z}|\mathbf{x}) = \mathcal{N}(\mathbf{z}; \boldsymbol{\mu}, \mathrm{diag}(\boldsymbol{\sigma}^2))$$
     - 使用重参数化技巧采样 $K$ 个隐变量：  
       $$\mathbf{z}^{(k)} = \boldsymbol{\mu} + \boldsymbol{\sigma} \odot \boldsymbol{\epsilon}^{(k)},\quad \boldsymbol{\epsilon}^{(k)} \sim \mathcal{N}(\mathbf{0}, \mathbf{I})$$
  2. **分布推前映射**：
     - 引入可学映射 $g_\phi : \mathbb{R}^{C \times D} \to \mathbb{R}^{L \times C}$（实现为两层 MLP，通道独立），得到预测样本：  
       $$\hat{\mathbf{y}}^{(k)} = g_\phi(\mathbf{z}^{(k)})$$
     - 预测分布定义为推前测度：$q_\phi(\mathbf{y}|\mathbf{x}) = (g_\phi)_\# p_\theta(\mathbf{z}|\mathbf{x})$。
  3. **混合目标优化**：
     - 使用高斯核密度估计 (KDE) 从样本 $\{\hat{y}^{(k)}_{c,t}\}$ 近似边际密度：  
       $$\hat{q}_h(y_{c,t}|\mathbf{x}) = \frac{1}{\sqrt{2\pi}K h}\sum_{k=1}^{K}\exp\!\Bigl(-\frac{(y_{c,t} - \hat{y}^{(k)}_{c,t})^2}{2h^2}\Bigr)$$
     - 对数域稳定计算，并做截断 $\tilde{q}_h = \max(\hat{q}_h, \varepsilon)$。
     - 最大似然损失：  
       $$\mathcal{L}_{\text{NLL}} = -\frac{1}{C L}\sum_{c=1}^{C}\sum_{t=1}^{L} \log \tilde{q}_h(y_{c,t}|\mathbf{x})$$
     - 辅助均值 MSE 损失：  
       $$\mathcal{L}_{\text{MM}} = \Bigl\| \mathbf{y} - \frac{1}{K}\sum_{k=1}^{K}\hat{\mathbf{y}}^{(k)} \Bigr\|^2$$
     - 最终损失：$\mathcal{L}_{\text{total}} = \alpha \cdot \mathcal{L}_{\text{NLL}} + \mathcal{L}_{\text{MM}}$（$\alpha=0.1$）。
- **理论支撑**：
  - 有限样本下 NLL 误差可分解为平滑偏置与有限 $K$ 波动项，给出了收敛性与超参数 $K,h$ 的权衡（定理5.1）。
  - 前推分布族在 $W_1$ 度量下可逼近任意真实条件分布（定理5.2）。
  - 梯度分析表明 NLL 项塑造分布形状，MM 项提供密集的一阶矩锚定，稳定训练。
