## 摘要
我们提出了首个神经概率幅度整形方法，该方法在考虑所有实现损耗的情况下优于现有方法。它采用了一种无块、易于实现的序列自回归编码器，该编码器与算术分布匹配兼容，从而降低了速率损失并提高了可达到的信息速率。

## Abstract
We present the first neural probabilistic amplitude shaping that outperforms existing methods while accounting for all implementation losses, using a block-less, easily implementable sequential autoregressive encoder compatible with arithmetic distribution matching, yielding reduced rate loss and higher achievable information rates.

### 1. 论文的核心问题与整体含义（研究动机和背景）

*   **研究背景**：概率幅度整形（PAS）是提升光纤通信系统非线性容忍度的关键技术，但有限长度的分布匹配器会引入速率损失（rate loss）。现有研究表明，即使在相同的边际符号分布下，符号的时间顺序也会显著影响非线性失真，这促使人们通过序列选择或神经整形来学习更好的联合符号分布。
*   **现有问题**：现有的神经概率幅度整形（NPAS）方法采用分块（blockwise）循环神经网络，存在三个主要缺陷：1）人为的分块边界忽略了跨块的符号相关性；2）块内逐渐增长的上下文导致非平稳的符号统计特性；3）更重要的是，训练过程没有显式考虑由符号依赖性造成的速率损失，导致高度复杂的时序结构可能带来严重的速率损失，从而抵消甚至逆转非线性整形增益。
*   **核心目标**：本文旨在提出一种全新的神经整形框架，通过序列化、无块的自回归模型和速率损失感知的训练目标，首次在考虑所有现实损耗的情况下，实现超越现有所有方法的性能。

### 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

*   **速率损失感知的训练目标 (Rate-Loss-Aware Objective)**：
    *   作者首先推导了考虑速率损失的优化目标，即在最大化可达信息速率（AIR）的同时最小化速率损失 $R_{loss}$。速率损失被定义为符号边际熵之和与联合过程熵率之间的差值：$R_{loss} = \frac{1}{N} \sum_{t=1}^{N} H(b_t) - \frac{1}{N} H(b_1, \dots, b_N)$。
    *   基于神经整形框架，提出了新的损失函数 $L^{++}$，其公式为：
        $$L^{++} = L + R_{loss} + \lambda D_{KL}(p(x) \| p_{MB}(x))$$
        其中 $L$ 是传统的基于交叉熵的速率损失项，$R_{loss}$ 是显式加入的速率损失惩罚，最后一项是KL散度正则项，用于鼓励学习到的边际分布 $p(x)$ 接近一个预优化的麦克斯韦-玻尔兹曼（Maxwell-Boltzmann, MB）目标分布，$\lambda$ 用于平衡两者。

*   **序列神经概率幅度整形编码器（Seq-NPAS）**：
    *   **核心思想**：将无符号符号序列建模为一个具有有限记忆的平稳自回归信源，如同“预测下一个词元”的语言模型。
    *   **模型结构**：采用单层Transformer解码器（带8个注意力头、隐藏维度64、前馈层维度256），结合旋转位置嵌入（Rotary Positional Embedding）和Swish门控线性单元（Swish-gated linear unit）以捕捉相对时序关系。
    *   **序列模型**：其联合概率分布公式为：
        $$p_{\theta}(a) = p_{\theta}(a_1^{\mu-1}) \prod_{t=\mu}^{N} p_{\theta}(a_t | a_{t-1}^{t-\mu+1})$$
        $\mu$ 是条件记忆长度（本文设为15）。在每个时间步，编码器根据先前生成的符号预测下一个符号的概率。这种序列化、无块的设计天然与算术分布匹配（ADM）兼容，并能够产生平稳的符号统计特性。

*   **训练与部署**：
    *   训练时，使用Gumbel-Softmax重参数化技巧和直通估计器进行可微分采样，并通过一个可微分的近似信道（单信道、单偏振扰动模型）进行端到端优化。
    *   部署时，训练好的模型参数作为ADM的概率表，用于生成符合学习到的联合分布的长序列，然后在真实WDM光纤信道中进行传输评估。
