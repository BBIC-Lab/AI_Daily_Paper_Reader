## 摘要
深度神经网络在医学图像分类中取得了优异性能，但往往如同黑箱运行。常用的事后解释方法通常提供启发式可视化，这些可视化与分类器预测分布的关系是间接的。本文引入了一种基于训练后分类器的输入依赖费舍尔信息矩阵（iFIM）的局部敏感性分析框架。iFIM刻画了在输入图像微小扰动下分类器预测分布的变化。通过使用Gram矩阵公式，可以在不显式构建完整图像维度的费舍尔矩阵的情况下恢复iFIM的非零特征谱。然后利用主导的iFIM特征空间将输入图像投影到高局部敏感性分量及其正交补集上。这些分量提供了模型内在的局部预测敏感性描述，而非传统的逐像素归因热图或任务相关解剖结构的因果分割。该框架在受控和临床医学图像分类任务上使用多种分类器架构进行了评估。基于扰动的实验表明，相较于低敏感性的互补分量，高敏感性iFIM分量与预测置信度和分类性能的变化具有更强的耦合。结果支持iFIM框架作为分析局部决策敏感性和补充医学成像中现有基于归因的可解释性方法的一种原则性工具。

## Abstract
Deep neural networks have achieved strong performance in medical image classification, but often work like black-box. Commonly used post-hoc interpretation methods often provide heuristic visualizations whose relationship to the classifier's predictive distribution is indirect. This work introduces a local sensitivity analysis framework based on the input-dependent Fisher Information Matrix (iFIM) of a trained classifier. The iFIM characterizes how the classifier's predictive distribution changes under infinitesimal perturbations of the input image. By using a Gram-matrix formulation, the nonzero eigenspectrum of the iFIM can be recovered without explicitly forming the full image-dimensional Fisher matrix. The leading iFIM eigenspace is then used to project an input image into a high local-sensitivity component and its orthogonal component. These components provide a model-intrinsic description of local predictive sensitivity, rather than a conventional pixel-wise attribution heatmap or a causal segmentation of task-relevant anatomy. The framework is evaluated on controlled and clinical medical image classification tasks using multiple classifier architectures. Perturbation-based experiments show that high-sensitivity iFIM components are more strongly coupled to changes in predictive confidence and classification performance than lower-sensitivity complementary components. The results support the iFIM framework as a principled tool for analyzing local decision sensitivity and for complementing existing attribution-based interpretability methods in medical imaging.

## 一、论文核心问题与研究动机

- 深度神经网络在医学图像分类中表现优异，但其内部决策过程往往呈**黑箱特性**，难以解释。
- 常用的事后可解释性方法（如 Grad‑CAM 系列、各种显著性图）多为**启发式可视化**，它们与分类器预测分布之间的关系是**间接且不严格**的，不同架构或实现细节可能导致不一致的结果。
- 因此，作者旨在提出一种**模型内在、原则性的局部敏感性分析工具**，直接从分类器的预测分布出发，刻画**输入微小扰动如何影响模型输出**，从而补充现有归因方法。

## 二、方法论

### 2.1 核心思想

- 对于训练好的概率分类器 $p_\theta(y\mid x)$，定义 **输入依赖费舍尔信息矩阵 (iFIM)**：
  $$F(x) = \mathbb{E}_{y\sim p_\theta(\cdot|x)}\Big[\nabla_x \log p_\theta(y|x)\,\nabla_x \log p_\theta(y|x)^\top\Big]$$
  该矩阵定量描述了预测分布在输入 $x$ 处的局部敏感度。
- 利用 iFIM 的特征结构，将输入图像投影到**高敏感性子空间**和**正交补空间**，得到任务敏感分量 $x_{sens}$ 和任务不敏感分量 $x_{comp}$。
- 这种分解不是传统的像素级归因热图，而是一种**局部预测敏感度的几何表征**。

### 2.2 Gram 矩阵分解（避免构造全尺寸 $F(x)$）

- 对图像输入 $x\in\mathbb{R}^d$，直接构建 $F(x)$ 需要 $O(d^2)$ 的存储和运算，对图像尺寸不可行。
- 构造矩阵 $G(x)\in\mathbb{R}^{d\times C}$，其第 $c$ 列为 $\sqrt{p_\theta(y=c\mid x)}\,\nabla_x\log p_\theta(y=c\mid x)$。
- 则 $F(x)=G(x)G(x)^\top$，其非零特征值与 **Gram 矩阵** $K(x)=G(x)^\top G(x)\in\mathbb{R}^{C\times C}$ 相同。
- 计算流程：
  1. 对每个类别 $c$ 计算梯度 $g_c(x)=\nabla_x \log p_\theta(c\mid x)$（需 $C$ 次反向传播）。
  2. 形成 $K(x)$，复杂度 $O(d C^2)$，特征分解 $O(C^3)$（对医学图像分类任务，$C$ 通常很小，如二分类时 $C=2$，秩为 1）。
  3. 由 $K(x)$ 的特征向量 $\{v_i\}$ 恢复 $F(x)$ 的对应特征向量：$u_i = G(x)v_i/\sqrt{\lambda_i}$。

### 2.3 图像分解

- 取 $F(x)$ 的非零特征值 $\lambda_1\ge\lambda_2\ge\dots\ge\lambda_R>0$，并按能量比 $\rho$ 选择前 $K$ 个特征向量：
  $$\frac{\sum_{i=1}^K \lambda_i}{\sum_{i=1}^R \lambda_i} \ge \rho$$
- 高敏感投影矩阵 $P_{sens}=U_K U_K^\top$，互补投影 $P_{comp}=I-P_{sens}$。
- 分解图像：
  $$x_{sens}=P_{sens}x,\qquad x_{comp}=x-x_{sens}$$
  两者正交；$x_{comp}$ 可能包含较低敏感度的非零特征方向和零空间。

### 2.4 局部扰动解释

- 对微小扰动 $\delta$，预测分布的 KL 散度近似为：
  $$\mathrm{KL}\big(p_\theta(\cdot|x)\,\|\,p_\theta(\cdot|x+\delta)\big) \approx \frac12 \delta^\top F(x)\delta$$
- 因此 $F(x)$ 的较大特征值对应的方向对预测的变化贡献最大，这一性质严格基于信息几何。

## 三、实验设计

### 3.1 数据集与任务

- **视网膜 OCT 图像分类**：三类（正常、玻璃疣、CNV），图像尺寸 $256\times256$，训练集每类 2000 张，验证/测试集各 100 张/类。
- **胸部 X 光心脏肥大检测**：二分类，图像尺寸 $1024\times1024$，训练集每类 800 张，验证/测试集 50/70 张。
- **受控 MRI SKS/BKS 仿真任务**：已知信号、已知背景，信号幅度 0.1–0.35 随机嵌入，图像尺寸 $256\times256$，总样本 11130 张（训练/验证/测试按比例划分）。

### 3.2 模型与对比方法

- **分类器架构**：VGG16、ResNet18，以及一个浅层 CNN（用于受控 MRI 任务）。
- **对比的事后可解释性方法**：DeepSHAP、Eigen‑CAM、Grad‑CAM、Grad‑CAM++、Score‑CAM、XGrad‑CAM。

### 3.3 评估协议

- **对比像素扰动实验**：分别从 iFIM 高敏感分量（取绝对值最大的 $k$ 个像素）和低敏感分量（取绝对值最小的 $k$ 个像素）中注入高斯噪声，扰动比例 $\alpha=1\%,2\%,3\%$，比较 F1 分数的比值（低敏感扰动后的 F1 / 高敏感扰动后的 F1）。比值越大，说明高敏感像素对分类的影响越大。
- **对抗扰动相关性分析**：用 FGSM 生成对抗样本 $x_{adv}$，计算原始图与对抗图的可解释性图之间的 Pearson 相关系数，衡量不同方法对决策改变扰动的稳定性。
- 所有实验均用 10 次随机初始化，报告均值与标准差。

## 六、主要结论与发现

- 高敏感性 iFIM 分量在受控 MRI 任务中准确集中于已知信号区域，在 OCT 和胸部 X 光任务中集中在病理相关区域（如黄斑病变层、扩大的心影轮廓）。
- 对比像素扰动实验显示，**iFIM 高敏感分量的像素被扰动后，F1 下降最为显著**，且在所有方法和任务中均取得最高比值（即低敏感扰动 F1 / 高敏感扰动 F1 最大），表明 iFIM 更好地区分了局部敏感与不敏感的像素。
- 对抗扰动下，**iFIM 高敏感图与原始图的相关性最低**（在 OCT 和 X 光任务中接近 0），说明该表征对改变决策的小扰动响应更剧烈，进一步证实其刻画了局部预测敏感度。
- 不同架构（VGG16 vs ResNet18）即使性能相近，其 iFIM 分解在空间分布上亦有差异，说明该框架能够揭示**架构依赖的局部行为差异**。
- iFIM 分析框架可作为**补充工具**，与常规归因方法并行使用，而非直接替代。

## 七、优点

- **原则性强**：直接从分类器的预测分布推导，基于费舍尔信息几何，有明确的理论支撑（局部 KL 散度近似）。
- **计算可行**：利用 Gram 矩阵回避 $d\times d$ 矩阵，仅需 $C$ 次梯度计算，复杂度与类别数相关，适合医学图像常见的小类别数任务。
- **模型不可知**：只需模型输出对输入的梯度，可应用于任意可微分类器，无需改动架构。
- **分解天然正交**：提供内在的正交投影，而非启发式的像素重要性排序。
- **能够揭示架构行为差异**：在相同性能的模型上，iFIM 可暴露不同的局部敏感性模式，有助于模型诊断。

## 八、不足与局限

- **局部性限制**：分析仅对 $x$ 附近微小扰动有效，无法全面描述分类器在较大扰动下的行为或全局特征重要性。
- **低敏感分量≠语义无关分量**：$x_{comp}$ 仅表示在当前模型的 iFIM 高敏感子空间之外，可能仍包含与任务相关但敏感度较低的信息，不宜直接解释为“不相关解剖结构”。
- **超参数依赖**：能量阈值 $\rho$ 的选择会影响分解维度，但文中未进行系统的消融分析或给出选择指南。
- **实验覆盖有限**：仅有 3 个任务、两种主架构，未在更多模态（如病理切片、超声）和最新架构（视觉 Transformer）上验证。
- **缺乏临床语义验证**：未与放射科专家标注的病灶分割进行定量对比，无法声称 iFIM 分量与真实病理区域的对齐程度。
- **计算开销未量化**：虽理论上避免了全尺寸矩阵，但未报告实际推理或分解的耗时，对实时部署的适用性不明。
