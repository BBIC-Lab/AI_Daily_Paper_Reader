## 摘要
实时过程监测需要能够从高维时间序列数据中提取可操作信息的方法。在本工作中，我们提出了一种结合拓扑数据分析（TDA）工具与机器学习的过程监测新方法。在所提出的方法中，我们将多元时间序列数据表示为流形，并使用拓扑描述符来概括此类数据的结构；然后我们使用神经常微分方程来学习系统拓扑结构的动态演化。通过使用来自工业过程的真实数据，我们表明这种基于轨迹的事件检测方法能够有效检测多种类型的事件。我们将该方法与基于重建的方法（如主成分分析和自编码器）以及基于轨迹的方法（使用Koopman自编码器）进行了对比。

## Abstract
Real-time process monitoring requires methods that extract actionable information from high-dimensional time-series data. In this work, we present a new approach for process monitoring that combines tools of topological data analysis (TDA) and machine learning. In the proposed approach, we represent multivariate time-series data as manifolds and use topological descriptors to summarize the structure of such data; we then use a neural ordinary differential equation to learn the dynamic evolution of the topological structure of the system. Using real data from an industrial process, we show that this trajectory-based event detection approach is effective at detecting diverse types of events. We contrast this approach against reconstruction-based approaches such as principal component analysis and autoencoders and against a trajectory-based approach that uses Koopman autoencoders.

### 1. 核心问题与研究动机

- **背景与挑战**：化工过程产生大量高维、耦合、非线性的多变量时间序列数据，实时监测需从中检测“事件”（故障、扰动等）。现有方法大体分为“基于重建误差”的静态方法（如 PCA、自编码器）和“基于轨迹演化”的动态方法（如 Koopman 自编码器）。
- **核心痛点**：重建类方法对主要事件（如停车、开车）敏感，但对次要、微弱的动态不稳定事件（如温度波动、速率变化）不敏感；动态方法虽引入时间结构，但实际检测能力仍集中在主要事件上，难以有效捕捉次要事件。
- **研究目标**：提出一种新的过程监测框架，能够同时有效检测主要与次要事件，并具备较好的泛化性和鲁棒性。

### 2. 方法论：TDA-NODE 框架

论文提出的方法将拓扑数据分析（TDA）与神经常微分方程（NODE）结合，组成“轨迹式”监测框架，核心流程如下：

- **流形表示**  
  对多变量时间序列数据按变量维和时间维排列成矩阵，经归一化后视为二维流形，每个像素值对应标准化后的过程变量值。使用滑动窗口（窗口长 1 天，重叠 20%）将全时段划分为若干窗口，每个窗口得到一个子流形 $Y_k \in [0,1]^{n \times M_k}$。

- **拓扑描述子提取**  
  在每个窗口上定义标量场 $h_k(i,j) = (Y_k)_{i,j}$，进行**子水平集滤波**：给定阈值 $\ell \in [0,1]$，构造子水平集  
  $$S_k(\ell) = \{ (i,j) \mid h_k(i,j) \le \ell \}$$  
  进而得到对应的立方复形，计算其**欧拉特性（Euler characteristic, EC）曲线**  
  $$\chi_k(\ell) = \beta_0^{(k)}(\ell) - \beta_1^{(k)}(\ell)$$  
  其中 $\beta_0$、$\beta_1$ 分别表示连通分量和 1 维洞的 Betti 数。取 $r$ 个阈值采样，得到拓扑潜变量 $z_k = [\chi_k(\ell_1), \dots, \chi_k(\ell_r)]^\top \in \mathbb{R}^r$，相当于窗口 $k$ 的 EC 曲线离散化。

- **拓扑动态学习（NODE 模型）**  
  将连续窗口的 EC 潜变量视为拓扑状态的连续演化，用 NODE 建模其动力学：  
  $$\dot{z}(t) = f_\theta(z(t))$$  
  其中 $f_\theta$ 是一个单隐层（64 神经元，tanh 激活）的前馈神经网络。$f_\theta$ 以当前窗口的 EC 向量 $z_k$ 为输入，预测其时间导数 $\dot{\chi}_k$。训练后，通过每一窗口的预测导数范数 $\|\dot{\chi}_k\|$ 的大小来检测事件：大幅跳变指示拓扑结构发生突然变化，即对应事件发生。

该方法具有以下特点：
- 拓扑描述子对噪声鲁棒，且具有缩放、旋转、变形下的不变性。
- 无需大量训练（TDA 部分无需训练），NODE 模型规模小，计算代价低。
- 利用的是拓扑状态的**方向性信息**，而非静态值，因此能捕捉到次要事件的细微动态。

- **静态 vs 动态**：基于重建误差的方法（PCA、AE）仅对主要事件有明显响应，次要事件几乎无可见峰；KAE 作为轨迹方法仍未有效检测次要事件。
