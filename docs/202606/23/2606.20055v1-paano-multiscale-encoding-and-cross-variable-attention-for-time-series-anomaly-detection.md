## 摘要
时间序列异常检测在工业与医疗监控等关键领域具有重要的实用价值。当前基于Transformer和大模型的检测方法会带来过高的计算开销，而现有的轻量级替代方案则受限于特征提取不足以及多变量间依赖关系建模的不充分。为克服上述缺陷，本研究在面向块的表示学习范式下，开发了一款轻量高效且名为PaAno的异常检测模型。编码器模块中，利用具有差异化感受野的卷积核构建多尺度特征提取骨干网络，以捕获分层时间特征；随后通过跨尺度自适应注意力聚合，并结合残差连接优化，进一步稳定特征表示学习。模型嵌入了跨变量融合注意力模块，显式刻画变量间相关性，使其能够在复杂运行条件下识别异常模式。此外，设计了一项基于时间片段窗口排序的新型预训练任务，以挖掘时间序列的内在结构属性，并利用三元组损失优化片段嵌入空间，增强特征判别力。在TSB-AD基准数据集上的大量实验表明，所提出的PaAno在单变量与多变量任务上均达到了最先进的检测精度，相较原始PaAno在包括VUS-PR在内的评价指标上取得了显著性能提升。通过紧凑的网络设计，该模型实现了较高的计算效率，可部署于资源受限的终端，进行实时异常推断。

## Abstract
Time-series anomaly detection has significant practical value for industrial and medical monitoring, as well as other critical domains. Current Transformer- and large-model-based detection approaches incur excessive computational overhead, while existing lightweight alternatives are constrained by insufficient feature extraction and inadequate modeling of dependencies across multivariate variables. To mitigate the above drawbacks, this study develops a lightweight, efficient anomaly detection model, dubbed PaAno, within the patch-oriented representation learning paradigm. In the encoder module, a multiscale feature-extraction backbone is constructed using convolutional kernels with differentiated receptive fields to capture hierarchical temporal characteristics; subsequent cross-scale adaptive attention aggregation, combined with residual connection optimization, further stabilizes feature representation learning. A cross-variable fusion attention module is embedded to explicitly characterize inter-variable correlations, empowering the model to identify anomalous patterns amid intricate operational conditions. Moreover, a novel pretext task based on temporal patch-window sorting is customized to uncover intrinsic structural properties of time series, and triplet loss is leveraged to optimize the patch embedding space for enhanced feature discrimination. Extensive experiments on the TSB-AD benchmark demonstrate that the proposed PaAno achieves state-of-the-art detection accuracy on both univariate and multivariate tasks, yielding significant performance gains across evaluation metrics, including VUS-PR, relative to the original PaAno. Leveraging a compact network design, the presented model achieves favorable computational efficiency, enabling deployment on resource-limited terminals for real-time anomaly inference.

## 2. 方法论

PaAno+的核心思想是在基于块的表示学习范式下，构建一个更具判别力的特征空间，让正常模式形成紧致簇，而异常模式游离其外。方法论主要由三部分关键设计组成：

**A. 多尺度时序编码器**
该模块用于捕获短、中、长程的层次化时序特征。
*   **多分支并行卷积**：采用三路并行的卷积结构，卷积核大小分别为3、5、9，分别对应细粒度、中粒度和粗粒度的局部模式提取。
*   **跨尺度自适应注意力聚合**：将三路输出的特征进行拼接，通过一层全连接网络计算各尺度对应的注意力权重 $W_{attn，i}$，再通过加权求和实现自适应融合：
    $$X_{fused} = \sum_{i=1}^{3} W_{attn,i} \cdot X_{scale,i}$$
*   **残差连接优化**：对原始的各分支输出 $X_{scale,i}$ 进行线性映射得到 $X_{residual，i}$，并与融合特征 $X_{fused}$ 相加，形成最终表征：
    $$X_{final} = X_{fused} + \gamma \cdot X_{residual}$$
    其中，$\gamma$ 为可学习参数。此设计旨在保留原始信息，稳定梯度传播。

**B. 跨变量融合注意力模块**
该模块用于显式建模多变量时间序列中不同变量间的依赖关系。
*   **输入投影**：将输入特征 $X \in \mathbb{R}^{B \times D \times T}$ （B为批次大小，D为变量维度，T为时序长度）线性投影至隐空间得到 $X_{proj}$。
*   **多头自注意力计算**：基于 $X_{proj}$ 计算 Query、Key和 Value 矩阵（$Q,K,V$），并沿变量维度计算注意力权重矩阵 $A$，量化变量间相对重要性：
    $$A = \text{Softmax}\left(\frac{QK^\top}{\sqrt{T}}\right)$$
    其中 $A \in \mathbb{R}^{D \times D}$。
*   **特征融合**：利用注意力权重对 $V$ 进行加权得到 $X_{attn} = AV$，然后将其与原始投影特征 $X_{proj}$ 拼接，再通过一个全连接层输出最终特征：
    $$y = W_{out} \cdot \text{Concat}(X_{attn}， X_{proj})$$

**C. 时序块窗口排序预训练任务**
为增强编码器对时序动态的理解，设计了一个自监督预训练任务，要求模型恢复被打乱的块窗口的原始顺序。
*   **样本构建**：从训练集中采样一个包含 $T$ 个连续补丁的窗口，通过随机排列 $\pi$ 打乱其内部顺序。
*   **任务目标**：模型通过一个由两层MLP构成的排序头 $c_\theta$，对拼接后的块嵌入特征进行预测，输出一个 $T \times T$ 的 Logit 矩阵 $O$。其中，$O_{b,i,j}$ 表示批次 $b$ 中第 $i$ 个块被预测为处在第 $j$ 个时序位置的置信度。
*   **损失函数**：排序损失 $\mathcal{L}_{\text{ranking}}$ 采用交叉熵形式，监督模型学习预测打乱操作的逆排列 $\pi^{-1}$：
    $$\mathcal{L}_{\text{ranking}} = - \frac{1}{M_{\text{valid}}} \sum_{b=1}^{M_{\text{valid}}} \sum_{i=1}^{T} \log\left( \frac{\exp(O_{b,i,\pi^{-1}(i)})}{\sum_{j=1}^{T} \exp(O_{b,i,j})} \right)$$
    其中 $M_{\text{valid}}$ 为有效样本数。

**D. 联合损失函数与异常检测流程**
*   **训练目标**：整体损失函数为三元组损失和排序损失的加权组合：$\mathcal{L}_{\text{total}} = \mathcal{L}_{\text{triplet}} + \lambda \mathcal{L}_{\text{ranking}}$。
    *   $\mathcal{L}_{\text{triplet}}$ 通过拉近锚点与正样本（随机偏移操作生成）的距离、推远与负样本（批次内余弦距离最大者）的距离，来构建判别性嵌入空间。
*   **推理阶段**：将所有训练用块的嵌入特征通过K-Means聚类，选取少量质心构建紧凑的记忆原型库。测试时，计算每个测试块嵌入到其 $k$ 个最近原型的平均距离作为块级异常分数，再对一个时间点覆盖的所有块的分数求平均，得到该点的最终异常分数。
