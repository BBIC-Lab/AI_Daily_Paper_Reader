## 摘要
基于意图的推荐系统通过建模用户行为背后的潜在动机，在提升准确性和可解释性方面获得了显著关注。现有模型大多通过聚类或原型学习直接从用户序列中推导意图。然而，这些方法对序列质量敏感，需要预设意图数量，且缺乏明确的语义基础。这些问题导致意图集不完整且粗糙，限制了推荐效果。本文提出基于意图推荐的稀疏自编码器（SAERec），这是一种新颖的推荐模型，能够从文本语料中自动构建细粒度且可解释的意图空间来指导推荐。SAERec 不将文本视为辅助信号，而是将其作为高信息密度的证据用于意图构建。具体而言，我们首先利用稀疏自编码器（SAE）从大语言模型（LLM）的潜在空间中解耦和解释文本嵌入，从而提取一组全面的细粒度可解释意图，将意图相关语义与文本噪声分离开来。然后，对于每个用户，我们从该集合中检索相关意图作为先验来指导推荐。这些意图包括匹配用户当前兴趣的个人意图，以及捕捉跨用户共享的一般物品模式（如质量、价格）的公共意图。最后，为了将检索到的意图整合到序列建模中，我们提出了一种多分支注意力机制，该机制捕获时间依赖性并注入个人和公共意图信号，其后接一个自适应融合层来构建最终的用户表示用于推荐。在公开数据集上的大量实验证明了 SAERec 的优越性，它一致优于最先进的基线方法，同时提供了人类可理解的解释。

## Abstract
Intent-based recommender systems have gained significant attention for improving accuracy and interpretability by modeling the underlying motivations behind user behaviors. Most existing models derive intents directly from user sequences via clustering or prototype learning. However, they are sensitive to sequence quality, require presetting the number of intents, and lack explicit semantic grounding. These issues lead to an incomplete and coarse intent set and limit the effectiveness of recommendation. In this paper, we propose the Sparse Autoencoder for intent-based recommendation (SAERec), a novel recommender that automatically constructs a fine-grained and interpretable intent space from a textual corpus to guide recommendation. Rather than treating texts as side signals, SAERec leverages them as high information density evidence for intent construction. Specifically, we first extract a comprehensive set of fine-grained interpretable intents from the latent space of large language models (LLMs) by using a sparse autoencoder (SAE) to disentangle and interpret text embeddings, which isolates intent-related semantics from textual noise. Then, for each user, we retrieve relevant intents from this set as priors to guide recommendation. It contains personal intents matching a user's current interests and public intents capturing general item patterns shared across users (e.g., quality, price). Finally, to integrate retrieved intents into sequence modeling, we propose a multi-branch attention mechanism that captures temporal dependencies and injects both personal and public intent signals, followed by an adaptive fusion layer to construct the final user representation for recommendation. Extensive experiments on public datasets demonstrate the superiority of SAERec, consistently outperforming state-of-the-art baselines while providing human-understandable explanations.

### 1. 论文的核心问题与整体含义 (研究动机和背景)

现有的基于意图的推荐系统 (Intent-based Recommender Systems) 存在三大核心瓶颈：
- **对序列质量敏感且意图集粗糙**：现有方法多通过聚类或原型学习从用户交互序列中直接推导意图，当交互序列稀疏或嘈杂时，会导致意图表示不可靠，进而降低模型性能。
- **需预设意图数量且缺乏语义基础**：这类方法需要人为预先定义意图的个数，数量过少会忽视用户多样性，过多则引入噪声；同时，学习到的意图向量多为难以理解的数值潜变量，缺乏显式的语义解释，限制了其可解释性和实际应用。
- **跨用户意图难以对齐和泛化**：部分基于文本的方法虽然具备语义，但通常针对单个序列独立推理，缺乏跨用户的语义对齐，且调用 LLM 推理延迟高、成本大。

因此，本文旨在提出一种全新方案，**自动从文本语料中构建一个全面、细粒度且具备人类可理解语义的意图集，并将其作为可靠的先验知识注入推荐模型**，以解决意图集不完整、不可解释以及泛化能力弱的问题。

### 2. 论文提出的方法论

SAERec 框架包含三个核心阶段：**意图集构建、双层级意图检索、基于意图的序列学习**。

- **意图集构建 (Intent Construction)**：
  - **文本编码**：利用推荐感知型 LLM (如 P5 的编码器) 对用户评论进行编码，获得稠密的文本嵌入 $e_i^l$。
  - **语义解耦与提取**：引入 Top-K 稀疏自编码器 (Sparse Autoencoder, SAE) 将上述稠密嵌入投影到高维稀疏空间。该过程通过重构损失 $\mathcal{L}_{\text{SAE}} = \left\| \hat{\mathbf{e}}_{i}^{l} - \mathbf{e}_{i}^{l} \right\|_{2}^{2}$ 训练，强制编码器仅激活 `TopK` 个最显著的特征单元，从而促使每个特征维度学习到单一的语义方向，获得潜在意图向量集合 $\mathcal{C}$。
  - **语义解释与意图过滤**：为每个解耦后的向量 $c_m$ 基于互信息目标自动关联最具代表性的自然语言词汇，形成文本解释。随后，利用 LLM (如 Mistral-7B) 根据这些词汇总结每个向量的语义，并自动筛选出与推荐场景相关的、具有明确含义的意图向量，构成最终的意图集 $\mathcal{C}^*$。

- **双层级意图检索 (Dual-Level Intent Retrieval)**：
  - **个人意图检索 (Personal Intents)**：使用序列编码器 (如 SASRec) 对用户序列编码得到序列表示，并引入可学习映射矩阵将意图集投影到同一空间。通过计算余弦相似度，为每个用户检索出与当前兴趣最相关的 Top-S 意图，作为个性化先验 $\mathcal{C}_{per}^{(u)}$。
  - **公共意图检索 (Public Intents)**：基于指数移动平均 (EMA) 和批次激活频率，筛选出跨用户普遍关注的通用意图，如“质量”、“价格”等，作为稳定且可泛化的共享先验 $\mathcal{C}_{pub}$。

- **基于意图的序列学习 (Intent-based Sequence Learning)**：
  - **多分支注意力机制 (Multi-Branch Attentions)**：设计了三个并行的注意力分支以分别建模：个人意图、公共意图和序列时序依赖。通过将序列表示作为 Query，个人/公共意图集作为 Key 和 Value，计算意图感知的用户表示 $P_{per}^{(u)}$ 和 $P_{pub}^{(u)}$。
  - **自适应表征融合 (Adaptive Representation Fusion)**：提出一种自适应融合层，根据用户上下文动态学习各个分支的融合权重 $w$，将三元表示加权融合为最终的用户表征 $P_f^{(u)} = \sum_{i=1}^{3} w^{(i)} \circ P^{(i)^{(u)}}$，以平衡个性化、泛化性和时序一致性。
  - 最终，基于融合表征计算下一个项目的预测概率分布，并采用交叉熵损失 $\mathcal{L}_{\text{Rec}}$ 进行优化。

### 7. 优点
- **新颖的意图构建范式**：首次将稀疏自编码器应用于推荐系统，从文本语料中发现并分离出单语义的意图因子，将意图发现与用户交互序列解耦，突破了传统聚类方法的限制。
- **全流程可解释且自动化**：形成了“关联词语 -> LLM 总结 -> LLM 筛选”的自动化意图解释与过滤链路，无需人工标注即可获得具有明确语义的意图标签。
- **双层级意图检索机制**：创新性地引入“个人”与“公共”两种意图先验，分别建模用户的即时兴趣与跨用户的稳定通用模式，有效平衡了个性化与泛化性。
- **意图注入与自适应融合精巧**：设计的意图感知的多分支注意力与基于用户上下文的自适应融合层，使得模型能动态调整不同信号源的贡献，优于简单的直接注入或加和。
- **实验论证极为详实**：覆盖了多个数据集的性能、效率、鲁棒性、可解释性、敏感性等全方位评估，对比的基线模型全面且先进，论点极具说服力。

### 8. 不足与局限
- **对文本数据的依赖性**：框架的成功构建严重依赖于用户评论文本或物品描述的存在与质量。在评论极度稀疏或者完全缺失的纯隐反馈推荐场景下，该方法的有效性有待进一步验证。
- **LLM 过滤的偏差风险**：使用 LLM 对意图进行自动化评判和筛选，虽然高效，但可能引入 LLM 自身的偏好或认知偏差，可能会错误地采纳或丢弃部分实际有用但模式不常见的意图。
- **超参数**：虽然进行了敏感性分析，但 SAE 的目标稀疏度 K、意图检索数量 S 等关键超参数仍需要针对不同数据集进行调优，这可能增加模型部署的复杂度。
- **上游 LLM 开销**：尽管 SAE 训练高效且意图集可复用，但方法论依赖于一个预训练好的推荐感知型 LLM (如 P5) 来进行文本编码，这部分模型的训练或部署本身存在一定的算力和成本需求。
