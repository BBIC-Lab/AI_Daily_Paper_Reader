## 摘要
结构约束的事件序列生成依然具有挑战性，因为生成的路径必须保持转移可行性、时序顺序、终止性和属性一致性。在预测性过程监控中，这一挑战表现为全事件序列生成，而已有工作主要解决下一活动、剩余时间、结果和属性预测等子任务。本文提出图接地交叉注意力Transformer神经网络（GGATN）用于这一统一的预测性过程监控任务。GGATN使用全局过程图作为结构化活动记忆，通过Transformer自注意力将序列位置情境化，并通过图接地交叉注意力注入过程拓扑。与自回归解码不同，GGATN在单次传递中生成活动、时间戳、长度以及事件级和序列级属性，随后通过维特比风格的图约束解码得到可行路径并显式终止。在六个基准事件日志上的实验表明，其生成质量比局部指令提示的大语言模型基线更可靠。GGATN在序列相似性、Damerau-Levenshtein相似性、基于二元组的控制流相似性和持续时间分布上均取得强性能，同时保持零虚构活动和零序列级属性不一致。消融分析证实全局图编码器是一个稳定的结构先验。可解释性分析展示了图结构、序列上下文、反馈优化和约束解码如何塑造生成过程。

## Abstract
Structurally constrained event sequence generation remains challenging because generated paths must preserve transition feasibility, temporal order, termination, and attribute consistency. In predictive process monitoring (PPM), this challenge appears as full event sequence generation, whereas existing work mainly addresses component tasks such as next activity, remaining time, outcome, and attribute prediction. This paper proposes the Graph Grounded Cross Attention Transformer Neural Network (GGATN) for this unified PPM task. GGATN uses a global process graph as structured activity memory, contextualizes sequence positions through Transformer self attention, and injects process topology through graph grounded cross attention. Unlike autoregressive decoding, GGATN generates activities, timestamps, length, and event level and sequence level attributes in a single pass, followed by Viterbi style graph constrained decoding for feasible paths and explicit termination. Experiments on six benchmark event logs show more reliable generation quality than local instruction prompted LLM baselines. GGATN achieves strong performance on sequence similarity, Damerau Levenshtein similarity, bigram based control flow similarity, and duration distribution, while maintaining zero hallucinated activities and zero sequence level attribute inconsistency. Ablation analyses confirm the global graph encoder as a stable structural prior. Interpretability analyses show how graph structure, sequence context, feedback refinement, and constrained decoding shape generation.

### 2. 论文提出的方法论

论文提出了**图接地交叉注意力Transformer神经网络（GGATN）**，其核心在于将全局流程结构作为先验知识与序列动态建模相结合。整个架构分为五个主要阶段：

*   **全局图注意力编码器**：
    *   从事件日志中构建全局过程图 $\mathcal{G} = ( \mathcal{V}, \mathcal{E}, \mathbf{X}_e, A )$，其中节点为活动，有向边为转移关系，边特征包含转移频率 $\mathbf{f}_{uv}$ 和平均时间间隔 $\mathbf{d}_{uv}$。
    *   使用边感知的图注意力网络（GAT）对活动节点进行编码，通过 $\ell$ 层堆叠和多头注意力机制，学习结构感知的活动嵌入矩阵 $\mathbf{E}_{act}$。该矩阵作为后续操作的结构化“记忆库”。

*   **序列Transformer编码器**：
    *   输入序列（包含活动、时间、属性等）经过标准化和嵌入后，加入正弦位置编码 $\mathbf{P}$，得到初始表示 $\mathbf{H}^{(0)}$。
    *   通过 $M$ 层多头自注意力（MHA）和前馈网络（FFN）的Transformer编码器，生成上下文感知的序列表示 $\mathbf{H}_{seq}$。

*   **图接地交叉注意力模块**：
    *   这是架构的核心交互机制。以序列表示 $\mathbf{H}_{seq}$ 作为**查询（Query）**，图编码器得到的活动嵌入 $\mathbf{E}_{act}$ 作为**键值（Key-Value）**，执行交叉注意力：
        $$\mathbf{C} = \text{MHA}(\text{LN}(\mathbf{H}_{seq}), \mathbf{E}_{act}, \mathbf{E}_{act})$$
    *   通过一个可学习的门控机制 $g = \tanh(\gamma)$，将图结构信息注入序列表示：
        $$\mathbf{H}_{cross} = \mathbf{H}_{tmp} + \text{FFN}(\text{LN}(\mathbf{H}_{tmp})), \quad \mathbf{H}_{tmp} = \text{LN}(\mathbf{H}_{seq} + g \cdot \mathbf{C})$$

*   **单次生成与图约束解码**：
    *   **非自回归生成**：模型通过一次前向传播，并行预测所有位置的**活动logits**、**时间戳**、**长度**及**事件级和序列级属性**。
    *   **维特比图约束解码**：生成最终活动序列时，不完全依赖于位置独立的argmax，而是采用图约束的动态规划（Viterbi-style）算法，结合一元（unary）活动分数和二元（pairwise）转移分数，在全局流程图上搜索一条得分最高且结构可行的路径，确保转移合法并显式终止于EOS节点。路径优化目标为最大化联合得分：
        $$\max_{y_1,\dots,y_T} \left[ \sum_{t=1}^T w_{unary}s_t(y_t) + s_{start}(y_1) + \sum_{t=2}^T s_{pair}(y_{t-1}, y_t) + s_{pair}(y_T, \text{EOS}) \right]$$

*   **反馈优化机制**：模型可选的通过初始预测的活动分布 $\mathbf{P}^{(1)}_{act}$ 计算一个“软”活动嵌入，并将其反馈到隐层状态中进行微调，从而对位置级活动分布进行重校准。

*   **训练目标**：采用加权多任务损失函数，联合优化活动、时间、长度和各级别属性的预测误差。

*   **实验数据覆盖范围**：虽然覆盖了6个数据集，但它们仍主要来自传统的业务流程管理（BPM）领域。在其他类型的序列数据上的泛化能力需要进一步验证。

*   **模型的复杂性与超参数**：GGATN设计精巧但组件众多（图编码器、Transformer、交叉注意力、反馈模块、解码器），其训练和配置相对复杂，需要针对数据集调整部分超参数（如辅助损失权重、解码参数），这可能增加了在新场景中应用的工程成本。
