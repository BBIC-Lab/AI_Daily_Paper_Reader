## 摘要
皮层内脑机接口受到日常神经信号漂移的影响，这会使预训练解码器性能下降。现有的无监督自适应方法依赖于深度循环或对抗架构，这些架构对于植入式硬件计算成本过高。我们提出膜电位对齐（MPA），一种针对脉冲神经网络的测试时自适应方法，通过仅使用KL散度匹配膜电位分布，将预训练解码器重新对齐到漂移的记录信号上。通过将更新限制在低秩（LoRA）权重，MPA只自适应不到9%的参数。在一项持续超过一个月的非人灵长类动物伸展任务中，MPA取得了与最先进的NoMAD方法相当的性能，同时使用了更简单的架构和更精细的时间分辨率（4毫秒对20毫秒）。这些结果表明，基于SNN的高效测试时自适应是实现长期、无需重新校准的脑机接口的实用途径。

## Abstract
Intracortical brain-computer interfaces suffer from day-to-day neural signal shifts that degrade pretrained decoders. Existing unsupervised adaptation methods rely on deep recurrent or adversarial architectures that are too computationally expensive for implantable hardware. We propose Membrane Potential Alignment (MPA), a test-time adaptation method for spiking neural networks that realigns a pretrained decoder to shifted recordings by only matching membrane potential distributions via KL divergence. By restricting updates to low-rank (LoRA) weights, MPA adapts fewer than 9% of parameters. On a non-human primate reaching task spanning over one month, MPA achieves performance competitive with the state-of-the-art NoMAD method, while using a simpler architecture and finer temporal resolution (4 ms vs. 20 ms). These results show that efficient SNN-based test-time adaptation is a practical path toward long-term, recalibration-free brain-computer interfaces.

- **整体含义**：本文提出一种面向 **脉冲神经网络（SNN）** 的测试时自适应方法——**膜电位对齐（MPA）**，仅通过匹配内部膜电位分布就能恢复解码精度，为植入式低功耗硬件上的长期“免校准”脑‑机接口提供了实用路径。

## 2. 方法论
### 2.1 核心思想
- 在源 session（Day 0）上监督式预训练一个 SNN 解码器，将其膜电位**分布**作为“参考”。
- 在目标 session（Day k）测试时，只利用无标签的神经信号，通过最小化 **KL 散度** 让自适应后的 SNN 膜电位分布与源分布对齐，从而恢复学得的输入‑输出映射。

### 2.2 关键技术细节
- **SNN 神经元模型**：采用 **漏电整合发放（LIF）神经元**，并使用 **软重置**（soft reset），使膜电位分布更连续、更利于 KL 梯度驱动。
  - 膜电位更新：
    $$u_i(t) = \beta u_i(t-1) + I_i(t) - S_i(t-1)V_{\text{th}}$$
    其中 $\beta$ 为膜电位衰减，$I_i$ 为输入电流，$V_{\text{th}}$ 为阈值，$S_i$ 为上一时步的脉冲。
- **膜电位分布对齐（MPA）**：
  - 对隐层 $l$，收集所有样本和时间步的膜电位向量，建模为多元高斯：
    $$p(\mathbf{u}^{(l)}) = \mathcal{N}(\boldsymbol{\mu}^{(l)}, \boldsymbol{\Sigma}^{(l)})$$
  - 对齐损失为 KL 散度：
    $$\mathcal{L}_{\text{MPA}} = D_{\mathrm{KL}}\big(p_0^{(l)} \| p_k^{(l)}\big)$$
    其中 $p_0^{(l)}$ 是在 Day‑0 数据上冻结预训练模型得到的分布，$p_k^{(l)}$ 是自适应模型在 Day‑k 数据上的分布。
- **低秩自适应（LoRA）**：
  - 仅对 **第一隐层** 的权重矩阵 $W$ 添加低秩更新：
    $$W' = W + AB$$
    $A\in \mathbb{R}^{d_{\text{out}}\times r}, B\in \mathbb{R}^{r\times d_{\text{in}}}, r\ll \min(d_{\text{in}},d_{\text{out}})$。
  - $A$ 初始化为零，保持初始输出不变；只有 $A,B$ 被训练，其余层冻结。实验中使用秩 $r=8$，自适应参数少于总参数的 9%。

### 2.3 源‑目标 session 兼容性评估
为避免在完全不兼容的 session 上无效自适应，提出两个先验指标：
- **表征相似性**：计算通道对相关矩阵的上三角向量之间的 Pearson 相关系数，衡量通道间统计关系的保持度。
- **Grassmann 距离**：对1‑秒分箱的发放矩阵进行 PCA，取前10个主成分构成的子空间，计算子空间主角度平方和的平方根：
  $$d_G(U_0, U_k) = \sqrt{\sum_{i=1}^K \theta_i^2}$$
  距离越小表示低维流形旋转越小，解码器几何有效性越强。

## 3. 实验设计
- **数据集**：非人灵长类（Indy）伸手数据集 [17]，共 23 个 session（2016‑09‑15 至 2017‑01‑24），每 session 96 通道。
  - 仅使用运动活跃片段，编码为 4‑ms 时间箱，预测手部二维速度标签，同样 4‑ms 分辨率。
  - 数据划分：50% 训练、25% 验证、25% 测试（各 session 内）。
- **预训练**：在 Day‑0（indy_20161005_06）上进行 10 epoch 监督训练，采用 3 层全连接 SNN（每层 64 个 LIF 神经元），50‑时步窗口，时变加权 MSE 损失。
- **测试时自适应**：对目标 session 使用 MPA，无标签，训练 10 epoch，仅更新第一层的 LoRA 权重，验证 KL 损失选最优。
- **对比方法**：
  - **无自适应 SNN**（同架构，不做 MPA）。
  - **LFADS / NoMAD**[9]：当前最先进的 TTA 方法，基于循环变分自编码器，处理 20‑ms 时间箱。由作者复现并验证。
- **评估指标**：行为解码的 $R^2$ 分数，重复 5 个随机种子，报告均值±标准差。

## 6. 主要结论与发现
- MPA 在大多数目标 session 上**显著提升** SNN 解码性能：平均 $R^2$ 从 0.09 升至 0.33，在 9/10 个 session 上优于无适应。
- MPA 在多个 session 上**性能与 NoMAD 相当或更优**，且模型架构更简单（3 层 SNN vs. 深层 GRU‑VAE），时间分辨率更细（4 ms vs. 20 ms）。
- 当源‑目标 session 在两种兼容性指标上**同时出现大幅偏差**时，MPA 和 NoMAD 均会失败，表明指标可用于预先筛选合适的源 session。
- MPA 在**极少量数据**下即可实现有效自适应（如 1.28 s 对齐数据仍带来可观增益），有利于边缘‑设备部署。

## 7. 优点
- **计算高效**：SNN 的脉冲驱动计算 + LoRA 仅更新少量参数（<9%），适合低功耗植入硬件。
- **无监督对齐**：KL 散度匹配内部膜电位，无需任何行为标签，实现纯测试时自适应。
- **细粒度时序**：4‑ms 分辨率解码，保留了更多动态信息，优于常见的 20‑ms 方法。
- **可解释的失败预防**：结合表征相似性和 Grassmann 距离，可提前判断自适应是否可信。
- **低数据需求**：短至数秒的未标记数据即可生效，实用性强。

## 8. 不足与局限
- **数据集局限**：仅在一个非人灵长类动物（Indy）上验证，缺乏跨个体、跨任务的泛化证据，转移到人类 iBCI 仍有待检验。
- **任务单一**：只评估了自我节律的伸手运动解码，未涉及更复杂或更多自由度的行为（如手指控制、语音）。
- **失败案例**：在 Day 8 等严重不兼容 session 上完全失效，表明仅靠 MPA 无法处理极端的神经群体重映射或阵列移位。
- **依赖源 session 选择**：需要用一个合适的源 session 提供参考分布，若源 session 本身不佳，方法可能失效；文中虽然提出了兼容性指标，但阈值是事后经验划分，缺乏先验自动选择机制。

- **硬件部署未验证**：仅进行了 PyTorch 仿真，未在真实的神经形态硬件上验证能效与实时性。
