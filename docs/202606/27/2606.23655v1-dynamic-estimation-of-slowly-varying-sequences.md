## 摘要
我们考虑顺序逼近缓慢变化序列中每个元素的函数的问题，即序列中位置 i 和 i-1 的元素差异大小 α_i 很小的情况。近期关于隐式迹估计的工作表明，当 α_t 很小时，重用对过去序列元素的查询可以降低总体成本 [Dharangutte & Musco, NeurIPS 2021; Woodruff et al., NeurIPS 2022]。我们引入一个框架，将其推广到多种向量空间上的线性和非线性函数，得到了矩阵幂、谱密度、蒙特卡洛积分和偏微分方程（PDEs）边值问题的新序列估计结果。此外，我们开发了一种在该框架中使用的新算法，该算法根据 α_t 局部缩放估计预算，获得了更紧的路径长度风格变化界，形式为 O(∑_{i=1}^m α_i) 用于估计长度为 m 的序列的成本。这改进了之前的隐式迹估计界 O(m·max_i α_i) [Dharangutte & Musco, NeurIPS 2021]，后者通过使用最坏情况的 α_i 固定查询预算，因此对于罕见爆发的稳定序列效率低下。最后，虽然过往工作都假设 α_i 的界已知，我们展示了在某些情况下如何以（几乎）不增加成本的方式实时估计变化。总之，我们的框架使序列逼近工具箱通用且自适应，同时改进了动态迹估计的最先进保证。

## Abstract
We consider the problem of sequentially approximating functions of each element in a slowly-varying sequence, i.e. one where the magnitude $α_i$ of the difference between the elements at positions $i$ and $i-1$ is small. Recent work on implicit trace estimation shows that when $α_t$ is small, reusing queries to past sequence elements can reduce the overall cost [Dharangutte \& Musco, NeurIPS~2021; Woodruff et al., NeurIPS~2022]. We introduce a framework generalizing this to a variety of linear and nonlinear functions on diverse vector spaces, obtaining novel sequential estimation results for matrix powers, spectral densities, Monte Carlo integration, and a boundary value problem from partial differential equations~(PDEs). Furthermore, we develop a novel algorithm for use with this framework that locally scales the estimation budget with $α_t$, obtaining sharper path-length-style variation bounds of form $\mathcal O(\sum_{i=1}^mα_i)$ on the cost of estimating a sequence of length $m$. This improves upon the previous implicit trace estimation bound of $\mathcal O(m\cdot\max_iα_i)$ [Dharangutte \& Musco, NeurIPS~2021], which is achieved by fixing the query budget using the worst-case $α_i$ and is thus inefficient for stable sequences with rare bursts. Lastly, while all past work assumes a known bound on $α_i$, we show in certain cases how the changes can be estimated on-the-fly with (nearly) no added cost. In summary, our framework makes the sequential approximation toolkit general-purpose and adaptive while improving upon state-of-the-art-guarantees for dynamic trace estimation.

- 整体含义：本文旨在建立一个通用、自适应的序贯估计框架，将过往方法从迹估计推广到更多线性与非线性函数，并用更紧的路径长度风格界 $\mathcal O(\sum\alpha_i)$ 取代旧界，同时通过在线步长估计消除对先验上界的依赖。

## 2. 论文提出的方法论
- **核心思想**：将估计过程视为动态维护一个运行估计值，每一步仅估计残差部分（当前状态与阻尼后前一状态之差），并重用上一步的估计结果。为使误差保持在目标内，根据局部变化幅度 $\alpha_t$ 来缩放采样预算 $k_t$ 和阻尼因子 $\gamma_t$。
- **静态估计器要求（Well-Concentrated Estimator）**：假设存在无偏随机估计器 $E(v,k)$，其误差具有次指数（sub‑exponential）浓度性质，且方差参数 $\nu$ 和尺度参数 $\beta$ 随样本数 $k$ 增加而衰减，具体满足 $\nu \leq c_1\|v\|_V/\sqrt{k}$，$\beta \leq (c_2\|v\|_V+c_3)/k$。Hutchinson 迹估计器、蒙特卡洛积分等均满足该条件。
- **已知步长的自适应算法（Algorithm 1）**：
  - 初始步：按最坏情况设置采样预算 $k_1$，得到估计 $\tilde{L}_1$。
  - 每步 $t$：令 $\gamma_t = \min(1,\alpha_t)$，残差状态 $u_t = v_t - (1-\gamma_t)v_{t-1}$，局部采样预算 $k_t$ 取为 $\mathcal O\big( \frac{c_1^2\alpha_t}{\epsilon^2} + \frac{c_2\alpha_t+c_3}{\epsilon} \big)\log(m/\delta)$；估计残差 $\hat{L}_t = E(u_t, k_t)$；更新 $\tilde{L}_t = (1-\gamma_t)\tilde{L}_{t-1} + \hat{L}_t$。
  - 理论保证（Theorem 1）：全序列误差同时 $\le \epsilon$ 的概率 $\ge 1-\delta$，总样本复杂度为
    $$\mathcal O\!\left( \left(\frac{c_1^2}{\epsilon^2}+\frac{c_2}{\epsilon}\right)\log\frac{m}{\delta}\left(1+\sum_{t=2}^m\alpha_t\right) + \frac{c_3 m\log(m/\delta)}{\epsilon} \right)$$
- **未知步长扩展**：假设存在范数估计预言机 $N(v,k)$，每步用 $k_{\text{norm}}$ 次查询获得 $\alpha_t$ 的近似值 $\hat\alpha_t \ge 0.9\alpha_t$（失败概率 $\delta/m$），然后放缩为 $\tilde{\alpha}_t = \hat{\alpha}_t/0.9$ 代入算法。定理 2 表明，该两阶段过程仅额外增加 $\sum_{t} k_{\text{norm}}$ 总开销，且对迹估计而言，$k_{\text{norm}} = \mathcal O(\log(m/\delta))$ 与 $\epsilon$ 无关。
- **应用迁移**：
  - **迹估计**：$V$ 为矩阵空间、范数 $\|\cdot\|_F$，$L$ 为迹，在线步长估计通过 Hutchinson 估计算子 $\Delta_t^\top\Delta_t$ 的迹实现，开销极低。
  - **矩阵幂与谱密度**：利用幂差的可加性 $\|A_t^k - A_{t-1}^k\|_F \le k\alpha_t$ 化为迹估计，引入倍数因子 $k$；谱密度由 $K=\tilde{\mathcal O}(1/\epsilon)$ 个矩同时跟踪，总成本为 $\tilde{\mathcal O}\!\left(\frac{1}{\epsilon^2}\left(1+\frac{1}{\epsilon}\sum\alpha_t\right)\log\frac{m}{\delta}\right)$。
  - **蒙特卡洛积分与 Dirichlet 问题**：将函数空间范数改为 $L_2(p)$ 范数，利用有界随机变量的 Bernstein 不等式满足浓度条件，得到类似的 $\mathcal O\!\left(\frac{\log(m/\delta)}{\epsilon^2}(1+\sum\alpha_t) + \frac{m\log(m/\delta)}{\epsilon}\right)$ 界；Dirichlet 问题通过 Kakutani 定理转化为边界函数的随机游走积分，直接继承该结果。

## 3. 实验设计

## 8. 不足与局限
