## 摘要
解耦表示学习是实现鲁棒属性预测的强大范式。尽管近期方法处理了属性间的相关性，但隐藏相关性仍未被充分探索：在特定属性值下的数据呈现出与其他属性相关的潜在模式。为保留模式信息并实现解耦，我们联合发现模式并强制基于模式的条件独立性。然而，这两个模块间的相互依赖可能在朴素迭代下导致错误放大。我们提出协调式迭代模式发现解耦（CoDID），这是一个端到端框架，具有能适应模式数量演变的动态架构，以及通过元优化缓解错误放大的协调机制。实验结果显示其在多种任务上达到了最先进性能。

## Abstract
Disentangled representation learning is a powerful paradigm for robust attribute prediction. While recent methods address attribute correlations, hidden correlations remain underexplored, where data under the value of a certain attribute exhibit underlying modes correlated with other attributes. To preserve mode information and achieve disentanglement, we jointly discover modes and enforce mode-based conditional independence. Yet, the interdependency between these two modules may lead to error amplification under naive iterations. We propose Coordinated Disentanglement with Iterative mode Discovery (CoDID), an end-to-end framework featuring a dynamic architecture that adapts to evolving number of modes, and a coordination mechanism that mitigates error amplification via meta-optimization. Empirical results demonstrate the state-of-the-art performance on diverse tasks.

### 2. 论文提出的方法论
- **核心思想**：联合模式发现和基于模式的条件互信息（CMI）最小化。通过迭代聚类估计模式标签，并用其作为条件变量来强制表示子空间的独立性；同时引入元优化学习样本权重，以在 CMI 最小化过程中保留模式信息，并利用权重模式指导聚类分裂，从而减轻簇内相关性带来的误差放大。
- **关键技术细节**：
  - **动态架构**：当聚类数目发生变化时，仅重构与聚类标签相关的输入/输出层神经元，其余网络参数保持不变，以保持训练稳定性。
  - **加权条件互信息最小化**：在判别损失 $L_{wd}$ 中对边际表示的样本进行加权 $w$，权重由权重网络 $W$ 根据当前表示质量、判别损失及条件标签动态生成：
    $$L_{wd}= \mathbb{E}_{c_1}\left[\mathbb{E}_{(z_1,z_2)|c_1}[\mathrm{lbce}(D(z_1,z_2,c_1),1)] + \mathbb{E}_{z_1|c_1,z_2|c_1}[w\cdot\mathrm{lbce}(D(z_1,z_2,c_1),0)]\right]$$
  - **元优化协调机制**：
    - 权重网络 $W$ 的优化目标 $L_{meta}$ 旨在最小化加权 CMI 后表示在属性预测和重建上的损失，以保持信息量：
      $$\tilde{\phi}_f(\phi_w) = \phi_f - \beta\nabla_{\phi_f}L_{wd}(\phi_f,\phi_w)$$
      $$L_{meta}(\phi_w) = \mathbb{E}_x\left[\mathrm{hce}(C_1(\tilde{z}_1),a_1) + \lambda_{mr}\cdot\mathrm{hmse}(R(\tilde{z}_1,\tilde{z}_2),x)\right]$$
    - 子聚类网络 $S$ 利用权重模式的散度来细化子簇概率 $\hat{r}^{sc}$，并通过权重对齐损失 $L_{wa}$ 引导聚类分裂，缓解簇内相关性。
  - **非参数聚类**：采用 Dirichlet 过程高斯混合模型 (DPGMM) 在每次迭代中自适应地确定簇数，并通过 split/merge 操作演化聚类结构。
- **算法流程**：先预训练编码器与预测器；然后迭代进行 DPGMM 聚类更新簇标签与数目，并交替优化信息量损失、加权判别损失、元损失和子簇损失。
