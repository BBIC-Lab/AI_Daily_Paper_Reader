## 摘要
节点表示学习发展迅速，但大多数现有方法依赖于按数据集进行训练和超参数调优。这种针对特定数据集的优化源于设计能够跨不同图数据集泛化的可重用图模型的困难。在这项工作中，我们提出了Node4All，一种适用于任意图数据集且无需任何数据集特定优化的节点表示学习方法。Node4All建立在两个互补的思想之上。在架构层面，我们引入了通道图变换器（CGT），它使得单个固定参数化能够处理任意图数据集。在学习层面，我们提出了一种基于一系列合成图的自监督学习。这些组件共同实现了超越单个数据集的泛化，而这是现有架构和学习框架无法做到的。我们在25个基准测试上，对21个基线方法（涵盖监督和自监督方法）进行了广泛的节点分类评估。尽管所有基线方法都针对每个数据集进行了训练和优化，但一个统一的Node4All在所有数据集上应用时，在21个基线方法中取得了第5名的竞争排名。此外，Node4All通过适当的预测器支持单样本学习和上下文学习，并在这些设置中优于最近的图基础模型（GFM）。这些结果表明，Node4All不仅实现了跨任意图数据集的可重用性，而且在实践中仍是一种有效的解决方案。代码和模型检查点可在 https://github.com/dooho00/node4all 获取。

## Abstract
Node representation learning has advanced rapidly, yet most existing methods rely on per-dataset training and hyperparameter tuning. This dataset-specific optimization comes from the difficulty of designing reusable graph models that generalize across diverse graph datasets. In this work, we introduce Node4All, a node representation learner applicable to arbitrary graph datasets without any dataset-specific optimization.   Node4All is built on two complementary ideas. At the architectural level, we introduce the Channel Graph Transformer (CGT), which enables a single fixed parameterization to process arbitrary graph datasets. At the learning level, we propose a self-supervised learning based on a series of synthetic graphs. Together, these components enable generalization beyond individual datasets, which is infeasible with existing architectures and learning frameworks. We extensively evaluate Node4All on node classification across 25 benchmarks against 21 baselines, covering both supervised and self-supervised methods. Despite all baselines being trained and optimized for each dataset, a single Node4All, applied uniformly across the datasets, achieves a competitive ranking of 5th among 21 baselines. Moreover, Node4All supports one-shot and in-context learning with an appropriate predictor and outperforms recent graph foundation models (GFMs) in these settings. These results demonstrate that Node4All not only achieves reusability across arbitrary graph datasets, but also remains an effective solution in practice. Code and model checkpoints are available in https://github.com/dooho00/node4all.

### 2. 论文提出的方法论
Node4All的核心由架构和学习两部分构成，共同消除现有方法的三个局限性：固定特征维度（L1）、数据集特定优化（L2）和任务特定偏差（L3）。

- **核心思想**：
  - **架构层（解决L1）**：将节点特征视为多个独立的**特征通道**（channel），而非一个整体向量，对每个通道用共享模型处理，使模型能自然适应任意输入特征维度 $F$。
  - **学习层（解决L2和L3）**：不使用任何真实世界数据集，而是生成大量多样化的**合成图**进行**掩码自编码预训练**，让模型学到跨数据集的结构-特征通用模式，避免对特定数据集或标签空间的过拟合。

- **关键技术细节**：
  - **通道图（Channel Graph）与通道式处理**：
    - 定义：将图 $G=(X,A)$ 分解为 $F$ 个通道图 $G^{(c)} = (x^{(c)}, A)$，其中 $x^{(c)}\in\mathbb{R}^N$ 是第 $c$ 个通道的标量信号。
    - 通道式处理：用一个共享函数 $f$ 独立处理每个通道图，然后将所有通道的输出拼接：$\mathbf{Z} = [f(G^{(1)}) \| \dots \| f(G^{(F)})] \in \mathbb{R}^{N\times F}$。
  - **通道图变换器（CGT）**：作为 $f$ 的具体实现，专门为处理标量通道图设计，包含两大步骤：
    - **多跳令牌化（Multi-hop Tokenization）**：将节点标量特征通过线性映射扩展为 $d$ 维令牌 $\mathbf{T}^{(0)} = x\mathbf{w}_{\text{in}}$，然后在每一层 $l$，为令牌的每个坐标 $i$ 分配不同的感受野，通过 $i$ 跳图传播得到 $\hat{A}^i \mathbf{T}^{(l-1)}_{:,i}$，再通过轻量MLP混合，形成新的令牌序列 $\mathbf{T}^{(0)}, \mathbf{T}^{(1)}, \dots, \mathbf{T}^{(L)}$。
    - **Transformer令牌读出（Readout）**：在每个节点前插入可学习的总结令牌 $\mathbf{t}_{\text{sum}}$，将节点所有层的令牌序列输入 $R$ 层Transformer，自注意力聚合信息到 $\mathbf{t}_{\text{sum}}$ 后，线性投影回标量，得到最终的通道嵌入 $f(G)_i$。
  - **合成图生成**：采用三阶段生成过程 $p(G) = \int p(z)p(A|z)p(X|A,z)dz$。
    - **结构生成**：使用 Chung-Lu 风格模型，由期望平均度 $\bar{d}$ 和度异质性参数 $\sigma$ 控制，首先生成期望度 $w_u = \bar{d} \exp(\alpha_u)$，然后依 $p_{uv} = \min(1, w_u w_v / W)$ 独立采样边。
    - **特征生成**：先生成低秩语义矩阵 $\mathbf{S}\in\mathbb{R}^{N\times \bar{F}}$（$\bar{F}=\lceil \alpha F\rceil$ 控制内在维度），再通过随机正交矩阵 $\mathbf{W}$ 映射到高维特征空间 $\mathbf{X} = \mathbf{S}\mathbf{W}$；之后进行**随机传播**：$\mathbf{X} \leftarrow \big((1+\beta\alpha)\mathbf{I} - \alpha \hat{A}\big)^S \mathbf{X}$，引入与图结构对齐的变化，使特征平滑度涵盖真实数据集的范围。
  - **掩码自编码（带节点丢弃）**：
    - 在合成图上随机丢弃节点子集 $\tilde{V}$，编码器在剩余节点组成的诱导子图上计算嵌入 $\mathbf{Z}_{V_o}$，丢弃节点嵌入置为0。
    - 解码器接受包含0嵌入的全图结构，重建被丢弃节点的原始特征，损失为余弦重建损失：$\mathcal{L}_{\text{recon}} = \frac{1}{|\tilde{V}|} \sum_{i\in\tilde{V}} \big(1 - \cos( f_{\text{dec}}(\tilde{\mathbf{Z}}, A)_i, \mathbf{X}_i )\big)$。
    - 编码器和解码器均采用CGT，预训练完成后，Node4All即可作为通用节点表示学习器，为任意图输出表示。

### 6. 论文的主要结论与发现
- **跨数据集竞争性能**：单一固定Node4All模型，在25个数据集上统一应用，不进行任何训练或超参数调优，在21个基线中综合排名第5。尤其在无法进行大量调参时（Fixed-HP），Node4All性能排名第1，展示出极强的稳定性和通用性。
- **少样本与上下文学习优势**：在单样本学习中，Node4All + Ridge回归器在5个数据集中的4个上优于特征对齐的图基础模型（如SAMGPT）；在上下文学习中，Node4All + TabPFN在平均排名和多数数据集上超越全归纳GFMs（如NodePFN, GraphAny），且无需像NodePFN那样的数据集特定预处理。
- **合成图预训练是泛化关键**：在真实世界数据集上预训练CGT只能得到中等性能（第9名），远不如合成图预训练（第5名），说明合成图提供的多样性和可控性是跨数据集泛化的核心驱动。
- **通道式处理与CGT不可或缺**：直接用GNN处理通道图或修改令牌化/读出方式都会导致显著性能下降，验证了CGT专门设计的合理性。

### 7. 优点
- **真正的即插即用**：首次实现单个无调优模型在多样化的图数据集上获得有竞争力的结果，极大地降低了图表示学习的使用门槛和计算成本。
- **巧妙的架构设计**：通道图分解和坐标式多跳令牌化将“任意特征维度”这一难题完美解决，并增强了可解释性（可单独观察每个通道的结构化效应）。
- **系统性的学习框架**：将合成图生成与掩码自编码结合，通过控制结构多样性、内在维度和图-特征相关性，为图模型提供了一个有效的通用预训练范式。
- **实验非常扎实**：对比基线广泛（21个）、评测设定多角度（全数据、单样本、上下文学习）、消融充分，结论令人信服。
- **实用性强**：训练快、表示可缓存，且完全免除昂贵的超参数搜索，具有极高的实用价值。

### 8. 不足与局限
- **合成图生成策略相对简单**：目前主要基于Chung-Lu随机图和低秩特征+随机传播，可能无法完全捕捉真实图中复杂的社区结构（如随机块模型）、高阶模体等模式，存在改进空间。
- **任务类型局限**：工作仅聚焦于节点分类任务。对于图级别的任务（如图分类、分子属性预测）和边级别的任务（如链接预测），当前框架无法直接适用，需要架构和预训练目标的扩展。
- **推理时间开销**：由于需要对所有特征通道进行独立处理，当特征维度很高时（如数千维），推理时间可能较长（虽然与维度成线性且可缓存，但仍比经过降维的传统GNN慢）。
- **依赖特征重建假设**：自监督目标假设节点属性至少部分可从上下文预测，对于结构信息极弱或完全随机的图可能失效，尽管这在实际应用中不常见。
- **对比的绝对性能**：虽排名第5（71.55%），但在某些数据集上与经过精细调参的最优监督/自监督方法（Tuned-HP）之间仍存在差距，尚无法在所有情况下达到SOTA，但通用性收益通常远大于微小性能损失。
