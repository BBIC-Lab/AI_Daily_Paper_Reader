## 摘要
在多变量时间序列异常检测领域，面对日益增长的数据复杂性和跨多个时间尺度的复杂依赖性，传统方法常常难以同时捕捉时间动态特征和复杂的序列间相关性。为此，我们提出了一个创新框架 MSCENet，它利用先进的时空学习和多尺度学习技术来提高检测精度。MSCENet 包含一个细粒度时间卷积模块，通过扩张卷积捕获复杂的时间依赖性，从而能够检测短期和长期模式。此外，该框架将序列间关系建模为图结构，使用 Mixhop 图卷积自适应地捕获跨不同时间尺度的空间依赖性。为了支持稳健的异常检测，MSCENet 中的多尺度门控卷积模块通过门控机制融合空间和时间属性，有助于检测多个尺度上的细微变化。在真实世界数据集 SMD、PSM 和 SWaT 上的实验评估表明，它为复杂时间序列数据环境下的异常检测提供了一种适应性强且高性能的解决方案。

## Abstract
In the field of multivariate time series anomaly detection, against the backdrop of increasing data complexity and complex dependencies across multiple temporal scales, traditional methods often struggle to simultaneously capture temporal dynamic features and intricate inter-series correlations. To address this, we propose an innovative framework, MSCENet, which leverages advanced spatio-temporal learning and multi-scale learning techniques to enhance detection accuracy. MSCENet includes a fine-grained temporal convolution module that captures complex temporal dependencies through dilated convolutions, enabling the detection of both short- and long-term patterns. Additionally, the framework models inter-series relationships as a graph structure, using Mixhop graph convolutions to adaptively capture spatial dependencies across varying time scales. To support robust anomaly detection, the multi-scale gated convolution module in MSCENet integrates spatial and temporal attributes through gated mechanisms, facilitating the detection of subtle variations across multiple scales. Experimental evaluations on real-world datasets: SMD, PSM, and SWaT. It provides an adaptable and high-performance solution for anomaly detection in complex time series data environments.

### 2. 论文提出的方法论

MSCENet 的核心思想是将多变量时间序列建模为图结构，并分别在时间维度和特征（空间）维度上，于多个时间尺度下进行依赖关系的捕获与融合。其主要组件和技术细节如下：

*   **输入嵌入与残差连接**：
    *   原始输入数据 $X_T$ 经过标准化后，通过一维卷积映射到 $d_{\text{model}}$ 维空间，并加入**位置编码**和**全局时间戳嵌入**（可学习），得到初始嵌入表示 $X_{\text{Temb}}$。
    *   网络主体由多个 **MSCEBlock** 堆叠而成，每个块的输出都通过残差连接与输入相加：$X^l_T = \text{MSCEBlock}(X^{l-1}_T) + X^{l-1}_T$。

*   **多尺度时间分析**：
    *   利用**快速傅里叶变换**对输入数据进行周期性分析，计算出前 $k$ 个最强振幅对应的频率 $f_1, \dots, f_k$。
    *   将这些频率转换为周期长度 $s_i = T / f_i$，作为后续分析的主导时间尺度。
    *   根据这些尺度 $s_i$，将一维时间序列通过填充和重塑操作，转换为不同的二维张量 $X^i_{2D} \in \mathbb{R}^{d_{\text{model}} \times s_i \times f_i}$，为后续的二维卷积操作做准备。

*   **细粒度时间卷积**：
    *   该模块旨在从二维张量中提取精细的时序特征。它包含四个并行的卷积层：一个标准卷积和三个扩张卷积。
    *   这三个扩张卷积分别采用**中心差分卷积**、**水平差分卷积**和**垂直差分卷积**。标准卷积捕获强度级别信息，而差分卷积增强梯度级别信息，从而更全面地刻画时间模式。其操作可概括为 $X^i_{\text{FGTConv}} = X^i_{2D} * (\sum_{j=1}^{4} w_j K_j)$。

*   **图结构空间相关性学习**：
    *   将对应于尺度 $i$ 的二维张量通过线性变换映射到带有 $N$ 个变量的特征空间。
    *   通过两个可训练的矩阵 $W^1_i$ 和 $W^2_i$，计算出一个**自适应的邻接矩阵** $M_i = \text{SoftMax}(\text{ReLU}(W^1_i (W^2_i)^T))$，动态地学习变量间的依赖关系。
    *   采用**Mixhop图卷积**来捕获高阶的序列间空间依赖关系，其核心操作为 $H^i_{\text{out}} = \sigma\left(\mathop{\|}_{j\in \beta} (M_i)^j H^i\right)$，其中 $\beta$ 是邻接矩阵的幂次集合。

*   **多尺度门控卷积与时空特征融合**：
    *   首先将 FGTConv 和 GCN 的输出相加，得到融合特征 $X^i_{\text{combined}}$。
    *   对 $X^i_{\text{combined}}$ 应用核大小分别为 $1 \times 3$、$1 \times 5$、$1 \times 7$ 的一维因果卷积，提取多尺度特征。
    *   对每路卷积输出应用**门控Tanh单元**：特征图被均分为 $E$ 和 $F$ 两部分，输出为 $\hat{X}^i_{S_j} = \tanh(E) \odot \sigma(F)$，通过门控机制控制信息流。
    *   将不同尺度的门控特征在时间维度上拼接，经池化和线性变换后，与输入 $X^i_{\text{combined}}$ 进行残差连接并应用ReLU激活，得到最终输出。
    *   最后，利用由FFT振幅决定的权重，将所有 $k$ 个尺度的输出张量聚合起来。

*   **异常检测**：
    *   模型最后通过一个前馈网络将聚合特征 $\hat{Z}$ 重构回原始数据空间 $\hat{Y}_t$。
    *   测试时，基于重构误差计算异常分数：$\text{AnomalyScore}_t = \frac{1}{N} \sum_{n=1}^{N} \lVert \hat{Y}_t - Y_t \rVert^2$。分数超过阈值即判定为异常。
