## 摘要
线性注意力模型允许每个token有固定的状态大小和固定的计算量。然而，由于状态大小有限，与基于softmax注意力的transformer架构相比，线性注意力模型在长上下文回忆方面表现落后。增加线性注意力的状态大小可以提高回忆性能，但代价是更高的FLOPs。在这项工作中，我们引入了稀疏Delta记忆（SDM），这是一种通过使用稀疏寻址方案将门控线性RNN的隐藏状态扩展到数量级更高容量的架构。SDM通过将密集的键值外积替换为对大型显式记忆的稀疏读写来扩展Gated DeltaNet架构。我们表明，在等FLOPs约束和相同参数数量的情况下，更高的状态记忆容量显著提高了上下文学习和长上下文检索任务的性能。此外，通过学习SDM记忆的初始状态，从而将其用作参数记忆，我们表明该模型进一步改善了广泛的常识和推理任务。

## Abstract
Linear attention models allow a fixed state size and a fixed amount of compute per token. However, due to their limited state size, linear attention models fall behind in long-context recall compared to softmax-attention-based transformer architectures. Increasing the state size of linear attention improves recall performance but at the cost of higher FLOPs. In this work, we introduce Sparse Delta Memory (SDM), an architecture that scales the hidden state of gated linear RNNs to orders of magnitude higher capacity using a sparse addressing scheme. SDM extends the Gated DeltaNet architecture by replacing the dense key-value outer product with sparse reads and writes to a large explicit memory. We show that, under an isoFLOP constraint and with an identical number of parameters, a higher state memory capacity significantly improves performance on in-context learning and long-context retrieval tasks. Moreover, by learning the initial state of the SDM memory and therefore using it as a parametric memory, we show that the model further improves on a wide range of common-knowledge and reasoning tasks.

### 2. 论文提出的方法论
- **核心思想**：将Gated DeltaNet的稠密键值外积更新替换为对大型显式记忆表的稀疏读写，通过产品键记忆的寻址方式在维持FLOPs不变的前提下将状态容量增大至数千倍。  
- **关键技术细节**：  
  - **记忆结构**：维护一个显式的记忆表 $\mathbf{M}_t \in \mathbb{R}^{N \times d_v}$，$N$ 为槽位数，远大于GDN的 $d_{qk}$。  
  - **稀疏键选择**：写键 $\mathbf{k}'_t$ 和读查询 $\mathbf{q}'_t$ 通过外和 $\mathbf{k}'_{1,t} \oplus \mathbf{k}'_{2,t}$ 得到 $N$ 个分数，利用 $\text{top-}k(\mathbf{s}_1 \oplus \mathbf{s}_2)=\text{top-}k(\text{top-}k(\mathbf{s}_1) \oplus \text{top-}k(\mathbf{s}_2))$ 的特性避免材料化完整分数矩阵，复杂度为 $\mathcal{O}(\sqrt{N}\times d + W^2 + R^2)$。  
  - **门控Delta写入**：对每个选中的写槽位 $i \in \mathcal{I}_t^w$，使用遗忘门 $\alpha_t$ 和输入门 $\beta_t$ 进行更新：  
    $$
    \tilde{\mathbf{M}}_t[i] = \alpha_t \mathbf{M}_{t-1}[i],\qquad 
    \mathbf{M}_t[i] = \tilde{\mathbf{M}}_t[i] + \beta_t \, k^{(i)}_t \bigl( \mathbf{v}_t - \tilde{\mathbf{M}}_t[i] \bigr)
    $$
    未被选中的槽位保持不变。  
  - **稀疏读取**：对选中的 $R$ 个读槽位加权求和：$\mathbf{y}_t = \sum_{i\in\mathcal{I}_t^r} q^{(i)}_t \cdot \mathbf{M}_t[i]$。  
  - **可学习初始状态**：将 $\mathbf{M}_0$ 设为模型参数，使记忆可编码预训练知识，不增加推理FLOPs。  
  - **等FLOPs设计**：设 $W=R=d_{qk}$，使每token计算量 $\mathcal{O}((W+R)\cdot d_v)$ 与GDN的 $\mathcal{O}(d_{qk}\cdot d_v)$ 匹配；通过调整头数 $H$ 控制状态总大小，避免随模型宽度 $d$ 呈 $\mathcal{O}(d^3)$ 爆炸。

### 6. 论文的主要结论与发现
- **等FLOPs下SDM显著优于GDN**：在所有规模上训练损失更低，8B下甚至超越全注意力模型的NLL。  
- **长上下文性能大幅提升**：在RULER上，SDM比GDN平均准确率提升超10个百分点，部分子任务（如单针2/3、多查询）接近或超过全注意力。  
- **参数化记忆的有效性**：学习初始状态对SDM带来额外的常识推理增益，而对GDN几乎没有帮助，说明大容量记忆能有效储存预训练知识。  
- **稀疏读写自适应性强**：读写键的分布呈现差异，写分布更集中，读分布更均匀，且模型能根据读写预算动态调节槽位使用模式。

### 7. 优点
- **计算效率与状态容量的突破**：首次在等FLOPs下将线性RNN的状态大小扩展3个数量级以上，同时保持常数Token级计算。  
- **性能提升全面**：不仅在长上下文检索上获得巨大增益，在短上下文常识推理中也通过可学习记忆获得提升。  
- **设计简洁且易于集成**：可即插即用于现有混合注意力架构，无需修改训练主流程。  
- **充分的实验验证**：涵盖缩放定律、多种下游任务、详细消融和内存访问分析，结论扎实。

### 8. 不足与局限
- **内存占用较高**：状态大小可能接近甚至超过模型参数量（如8B模型状态约8GB），不适合资源极度受限的场景；虽然相比长序列KV缓存仍有优势，但内存敏感部署下需权衡。  
- **训练速度受限于稀疏核**：尽管FLOPs理论匹配，实际MFU较低，端到端训练吞吐量慢于GDN，需要更高效的稀疏算子。  
- **部分长上下文任务仍不足**：在RULER的 `multikey_2` 等多跳复杂检索任务上，固定状态模型性能远逊于全注意力，容量扩展虽缓解但未根本解决。  
- **超大规模未验证**：实验止于8B参数、约1T训练token，更大规模下的行为及与超长上下文微调的结合仍待探索。
