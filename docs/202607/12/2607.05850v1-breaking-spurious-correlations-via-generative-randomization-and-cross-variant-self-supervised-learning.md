## 摘要
使用经验风险最小化训练的深度神经网络在分布偏移下常常失败，因为它们利用了物体标签与背景上下文之间的虚假关联。最近的生成式方法通过生成背景改变的因果反事实图像来解决这个问题，但通常将这些样本作为标准数据增强使用，使得模型仍有可能保留背景敏感的表征。我们提出一个两阶段框架，利用生成式干预来显式地学习背景不变的表征。首先，我们使用零样本分割分离前景物体，并用结构保持扩散模型生成上下文变化的变体，在保持物体身份的同时改变周围环境。然后，我们引入了跨变体自监督学习，其中同一物体在不同背景下的变体在对比目标中构成正样本对。这鼓励编码器对齐以物体为中心的表征，同时抑制背景特定的线索。随后，我们使用经验风险最小化预热和逐层学习率的 GroupDRO 对预训练编码器进行微调。在分布偏移基准上的实验展示了最优的最差组性能，在 Waterbirds 上达到 92.5%，在 MetaShift 上达到 81.7%，在 NICO++ 上达到 87.4%。代码：https://github.com/surajyadav-research/GRSSL

## Abstract
Deep neural networks trained with Empirical Risk Minimization (ERM) often fail under distribution shifts because they exploit spurious correlations between object labels and background context. Recent generative approaches address this issue by creating counterfactual images with altered contexts, but typically use these samples as standard data augmentation, leaving the model free to retain background-sensitive representations. We propose a two-stage framework that uses generative intervention to explicitly learn background-invariant visual representations. First, we isolate the foreground object using zero-shot segmentation and generate context-shifted variants with a structure-preserving diffusion model, preserving object identity while varying the surrounding environment. We then introduce Cross-Variant Self-Supervised Learning, where variants of the same object under different backgrounds form positive pairs in a contrastive objective. This encourages the encoder to align object-centric representations while suppressing background-specific cues. Then, we fine-tune the pretrained encoder using an ERM warm-up followed by GroupDRO with layer-wise learning rates. Experiments on distribution-shift benchmarks demonstrate best worst-group performance, achieving 92.5% on Waterbirds, 81.7% on MetaShift, and 87.4% on NICO++. ### 2. 方法论
#### 核心思想
- 两阶段框架：先通过生成式干预制造背景随机化的训练变体，再通过跨变体对比学习(Cross-Variant SSL)学习背景不变表征，最后用分组鲁棒优化(GroupDRO)微调分类器。
- 关键点：不再将生成图像仅作为普通数据增强，而是利用“同一物体、不同背景”的变体构成自监督正样本对，强制编码器忽略背景。

#### 关键技术细节
- **第一阶段：生成背景随机化变体**
    - 使用零样本分割（如CLIPSeg）提取图像中的前景物体掩膜。
    - 采用结构保持的扩散模型，以原始图像为条件，结合物体掩膜进行修复式生成：保持前景物体区域完全不变，仅重绘背景区域，从而生成前景一致、背景变换的多个变体。
- **第二阶段：跨变体自监督学习**
    - 对每张训练图像，生成 $K$ 个背景变体，形成变体集合。
    - 构建对比学习目标：同一物体在不同背景下的变体互为正样本对，不同物体（或同一物体但背景相同的原始样本）为负样本对。损失函数鼓励编码器对同一物体但背景不同的变体输出相似的表征，对背景相似的样本则抑制相似性。此时表征训练不使用类别标签。
- **第二阶段后续：监督微调**
    - 对预训练好的编码器添加分类头，先进行少量Epoch的ERM热身训练（使用原始数据集，含生成变体作为数据增强？*注：原文称“结合ERM热身和分组鲁棒优化(GroupDRO)微调”*）。
    - 然后切换至GroupDRO损失，对不同子组（由物体类别与背景类别组合定义）加权训练，并采用逐层递减的学习率，以鲁棒地适配群体分布。

#### 公式与算法流程（文字描述）
整体流程：
1. 输入训练集 $D = \{(x_i, y_i)\}$。
2. 对每个 $x_i$，用分割模型提取前景物体掩膜 $M_i$；用扩散模型根据“保留 $x_i$ 中 $M_i$ 区域、重绘背景”生成 $\{x_i^{(1)}, \dots, x_i^{(K)}\}$。
3. 构建对比学习数据集：对于变体 $x_i^{(k)}$，其正样本为同一原图的其他变体 $\{x_i^{(j)}\mid j \neq k\}$，负样本为其他物体原图及其变体。通过 InfoNCE 类损失训练特征提取器 $f_\theta$，使 $f_\theta(x_i^{(k)})$ 接近 $f_\theta(x_i^{(j)})$，远离其他表征。
4. 在 $f_\theta$ 上添加分类头，先用 ERM 热身训练，再用 GroupDRO 最小化最差组的风险，优化目标为 $\min_\theta \max_{g \in \mathcal{G}} \mathbb{E}_{(x,y)\sim P_g}[\ell(f_\theta(x),y)]$，其中 $\mathcal{G}$ 为预定义的子组集合。
