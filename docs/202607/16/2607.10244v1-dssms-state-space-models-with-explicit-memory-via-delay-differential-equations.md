## 摘要
状态空间模型（SSM）已成为高效长序列建模的强大范式，提供并行训练和快速的线性时间循环推理。然而，与其他循环架构一样，SSM必须将无限历史压缩为固定大小的状态，这限制了上下文保留，并使得对长程上下文的精确检索变得极其困难。为了克服这一限制，我们提出了延迟状态空间模型（DSSM），这是一种受延迟微分方程（DDE）启发的对角SSM扩展，用显式的延迟状态反馈增强了离散SSM循环。实现实用的显式延迟反馈需要新的稳定性参数化、历史管理和快速傅里叶变换训练工具。我们通过一个简单的延迟无关稳定性条件为基础，采用实用的离散化和参数化来应对这些挑战。为了绕过直接时域核构建，我们推导了DSSM传递函数并在频域中计算核，通过核轮廓移位来抑制混叠并恢复准确的快速傅里叶变换训练。实验表明，DSSM显著改进了针对性的延迟检索任务，同时在大多数标准序列指标上优于S4D，并在其余指标上保持接近。

## Abstract
State Space Models (SSMs) have emerged as a powerful paradigm for efficient long-sequence modeling, offering parallel training and fast linear-time recurrent inference. However, like other recurrent architectures, SSMs must compress an unbounded history into a fixed-size state, which limits context retention and makes precise retrieval over long-range context inherently difficult. To overcome this limitation, we propose Delay State Space Models (DSSMs), a delay differential equation (DDE)-inspired extension of diagonal SSMs that augments discrete SSM recurrences with explicit delayed-state feedback. Making explicit delayed feedback practical requires new stability parameterization, history management, and FFT-training tools. We address these challenges with a practical discretization and parameterization grounded in a simple delay-independent stability condition. To bypass direct time-domain kernel construction, we derive the DSSM transfer function and compute kernels in the frequency domain, using a kernel contour shift to suppress aliasing and recover accurate FFT training. Empirically, DSSMs substantially improve targeted delayed-retrieval tasks while outperforming S4D on most standard sequence metrics and remaining close on the others.

- **核心问题**：如何在不牺牲 SSM 高效训练与推理优势的前提下，增强其对长距离信息的精确记忆与检索能力？
- **整体思路**：受延迟微分方程（DDE）启发，将时滞反馈显式地引入 SSM 的状态演变，使模型直接访问过去某个时刻的状态，从而突破纯 ODE 的记忆瓶颈。本文提出的延迟状态空间模型（DSSM）即在这一思路下，通过对角 SSM 的扩展，重新设计稳定性参数化、历史管理与频域训练方案，形成一套可实用化的端到端序列建模框架。

## 2. 方法论

### 2.1 连续时间延迟 SSM
- 连续 DDE 公式（单通道 SISO 系统）：
  $$\dot{h}(t) = A h(t) + A_d h(t-\tau) + B x(t), \quad y(t) = C h(t)$$
  其中 $A$ 为状态矩阵，$A_d$ 为延迟矩阵，$\tau$ 为可学习延迟，$x(t)\in\mathbb{R}$ 为输入，$y(t)\in\mathbb{R}$ 为输出。
- 为简化并保留对角结构，限制 $A$ 和 $A_d$ 为对角矩阵，系统解耦为 $N$ 个独立标量 DDE：
  $$\dot{h}_n(t) = \lambda_n h_n(t) + \mu_n h_n(t-\tau) + b_n x(t)$$

### 2.2 离散化与稳定性
- **历史缓冲与线性插值**：维护定长历史缓冲 $B_{t-1}$，包含最近的隐藏状态。由于连续时滞 $\tau$ 不一定落在离散时间网格上，将其分解为 $k=\lfloor\tau/\Delta t\rfloor$ 和分数 $\alpha\in[0,1)$，通过线性插值近似延迟状态：
  $$\tilde{h}_{t-\tau} = (1-\alpha)\hat{h}_{t-k} + \alpha\hat{h}_{t-k-1}$$（$\hat{h}$ 为填充后的缓冲状态）。
- **稳定性条件**：针对标量零输入系统 $h_t = \bar{\lambda}h_{t-1} + \bar{\mu}[(1-\alpha)h_{t-k} + \alpha h_{t-k-1}]$，给出延迟无关的渐近稳定充分条件：
  $$|\bar{\lambda}| + |\bar{\mu}| < 1$$
- **稳定参数化与离散化**：
  - 引入共享连续基 $\zeta_n$（$\mathrm{Re}(\zeta_n)<0$），经 ZOH 离散得到总记忆预算 $\bar{\zeta}_n = \exp(\zeta_n\Delta t)$，其模值 $|\bar{\zeta}_n|\in(0,1)$。
  - 利用可学习的门控 $p_n = \mathrm{sigmoid}(r_n)$ 分配预算给延迟分支：
    $$\bar{\lambda}_n = (1-p_n)\bar{\zeta}_n,\quad \bar{\mu}_n = p_n \bar{\zeta}_n e^{i\theta_n}$$
    从而保证 $|\bar{\lambda}_n| + |\bar{\mu}_n| = |\bar{\zeta}_n| < 1$，逐模满足稳定条件。
  - 输入投影沿用标准 ZOH 形式：$\bar{B} = (\bar{A} - I)A^{-1}B$，其中 $\lambda_n$ 可由 $\zeta_n$ 反推。
- 离散向量递归：
  $$h_t = \bar{A} h_{t-1} + \bar{A}_d \tilde{h}_{t-\tau} + \bar{B} x_t$$
  $\bar{A} = \mathrm{diag}(\bar{\lambda}_1,\dots,\bar{\lambda}_N)$，$\bar{A}_d = \mathrm{diag}(\bar{\mu}_1,\dots,\bar{\mu}_N)$。

### 2.3 频域训练与核轮廓移位
- 延迟反馈破坏了标准 SSM 的简单时域卷积核 $\bar{K}_t = C\bar{A}^t\bar{B}$，因此无法直接使用 FFT 训练。
- **传递函数推导**：在 Z 域推导单通道系统的严格传递函数：
  $$K(z) = c\left[1 - z^{-1}\bar{\lambda} - z^{-k}D(z)\bar{\mu}\right]^{-1}\bar{b}, \quad D(z) = (1-\alpha) + \alpha z^{-1}$$
- **抑制混叠的核轮廓移位**：由于延迟系统产生长尾回声，直接 IFFT 恢复的循环核会产生明显的时域混叠。通过引入阻尼因子 $\rho\in(0,1)$ 构造移位系统 $(\bar{\lambda}'=\rho\bar{\lambda},\bar{\mu}'=\rho^k\bar{\mu})$，使得线性核 $\bar{K}_t$ 变为 $\bar{K}'_t = \rho^t \bar{K}_t$，有效压制混叠，再通过 $\bar{K}_t \approx \rho^{-t}\tilde{K}'_t$ 恢复原始核。选取 $\rho^M \approx \varepsilon_{\text{shift}}$（如 $\varepsilon_{\text{shift}}=10^{-3}$）平衡混叠与浮点误差。
- **算法流程**（对每通道）：
  1. 在 FFT 网格 $z_m = \exp(i2\pi m/M)$ 上计算移位传递函数 $K'(z_m)$（逐模求和）。
  2. 对 $\hat{K}'$ 应用 IFFT 获得阻尼循环核 $\tilde{K}'$。
  3. 反阻尼恢复长度 $L$ 的线性核 $\bar{K}_t$ 并零填充至 $M$。
  4. 标准频域卷积 $y = \text{IFFT}(\text{FFT}(\bar{K}) \odot \text{FFT}(x_{\text{pad}}))$。
- **复杂度**：每通道 $O(M(N+\log M))$，全层带 $H$ 通道为 $O(HM(N+\log M))$。

### 2.4 架构与初始化
- 以对角 SSM（S4D）为骨干，构建 DS4D 层：$H$ 个并行通道，每个通道 $N$ 个对角状态模，通道间共享一个时滞 $\tau$。
- 基础动力学初始化继承 S4D-Inv 或 S4D-Lin。
- 延迟分支初始化：相位 $\theta_n$ 初始化为 $\omega + \epsilon_n$（$\omega$ 接近 0 或 $\pi$），以提供延迟增强（$\omega\approx0$）或变化敏感（$\omega\approx\pi$）的先验；门控 $p_n$ 从小值开始。
- 时滞初始化：一般任务使用均匀网格初始化 $q$（然后变换到 $\nu_\tau$），若任务有先验延迟长度（如行宽），则采用任务特设初始化。

## 6. 主要结论与发现

- DSSM 通过引入显式延迟状态反馈，极大增强了 SSM 在精确长程检索与延迟聚合任务上的能力，性能大幅超越 S4D、Mamba 等基线，并接近或持平 Transformer 基线。
- 在通用序列任务（sMNIST、SC、BIDMC、LRA）上，DS4D 在绝大多数指标上优于或持平 S4D，证明延迟机制并未损害普适的序列建模能力。
- 所提出的频域训练算法结合核轮廓移位，使得 DSSM 能像标准 SSM 一样保持高效的 FFT 并行训练，同时维持推理时的线性时间复杂度和近恒定解码吞吐。
- 参数学习过程表明，模型倾向于保留分布式延迟银行，而非坍缩为单一延迟，说明多通道延迟分布对应对不同时间尺度有益。

## 7. 优点

- **实验多样化且说服力强**：从纯净的合成任务发端，逐步验证在真实多模态和长程竞技任务上的表现，并从效率、精度、消融、可视化多角度佐证，论证链条完整。
- **严格的可比设计**：参数匹配、统一的训练配置、官方基线实现，减少了不公平比较。
- **清晰的局限坦诚度**：明确量化了解码内存的增长、FFT 算法等限制，为后续改进提供了明确方向。

## 8. 不足与局限

- **解码内存依赖历史窗口**：DSSM 需要缓存 $L_{\max}$ 长度的历史状态用于延迟访问，解码内存随窗口线性增长，无法做到纯 $O(1)$，在超长上下文（如百万级 token）场景受到约束。
- **时滞与分支初始化尚需系统化**：虽然实验中使用固定先验或均匀网格效果不错，但缺少理论指导如何根据任务自动选择 $\tau$ 和分支相位，目前更多依赖启发式。
- **未涉及选择性/输入依赖机制**：DSSM 当前是完全线性时不变系统，无法像 Mamba 那样根据输入动态调整参数，可能限制了在需要上下文依赖门控（如语言建模）的场景中的表现。
- **缺少大规模语言模型实验**：所有测试均在中等规模的分类或回归任务上完成，尚未验证在大规模自回归语言建模中是否仍能保持训练稳定性和性能增益。
- **频域训练的精度取舍**：核轮廓移位引入了 $\varepsilon_{\text{shift}}$ 超参数，虽在实验中表现良好，但较小的容差可能带来浮点误差放大，对极长序列或高精度要求可能需进一步优化。
- **延迟效应的理论理解不足**：虽然稳定性条件已解决，但对于延迟反馈如何塑造长程记忆、在不同任务中应如何分配延迟分布等，尚未深入定量分析。
