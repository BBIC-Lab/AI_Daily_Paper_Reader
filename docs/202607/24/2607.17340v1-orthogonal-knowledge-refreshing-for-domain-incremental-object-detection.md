## 摘要
领域增量目标检测（DIOD）要求模型在保留先前知识的同时不断适应新领域。近年来，参数高效微调提供了一个有前景的途径，其中冻结预训练模型并注入少量可学习参数用于下游任务。然而，这些方法存在覆盖关键过去知识的风险，会引发领域间干扰和性能退化。为应对这一挑战，我们提出了正交知识刷新（OKR），这是一个简单而有效的DIOD框架。OKR通过为每个领域指定专用的低秩分支来增量式地构建独立的领域特定子空间，这些子空间无缝融合以进行整体决策，从而在推理时无需领域选择即可实现无冲突的容量扩展。为最大限度地减少融合过程中的知识干扰，我们提出了一种基于梯度的正交刷新策略，将新领域的梯度更新投影到融合历史子空间的正交补上，支持无遗忘的持续适应。此外，为减轻跨领域的语义碎片化，我们强制拓扑感知一致性，对齐旧领域和新领域的语义结构。大量实验验证了OKR的优越性，在Pascal VOC和BDD100K系列数据集上分别以+5.6%和+6.5%的显著mAP提升超过了最佳无样本方法。

## Abstract
Domain-incremental object detection (DIOD) requires models to continually adapt to new domains while preserving prior knowledge. Recently, parameter-efficient fine-tuning offers a promising avenue, wherein a pre-trained model is frozen and a small number of learnable parameters are injected for downstream tasks. However, these methods risk overwriting critical past knowledge, triggering inter-domain interference and performance degradation. To address this challenge, we propose Orthogonal Knowledge Refreshing (OKR), a simple yet effective framework for DIOD. OKR incrementally constructs independent domain-specific subspaces via dedicated low-rank branches for each domain, which are seamlessly fused for a holistic decision, enabling conflict-free capacity expansion without domain selection during inference. To minimize knowledge interference during fusion, we present a gradient-based orthogonal refreshing strategy that projects gradient updates of new domains onto the orthogonal complement of the fused historical subspace, supporting continual adaptation without forgetting. Moreover, to mitigate semantic fragmentation across domains, we enforce topology-aware consistency, aligning the semantic structures of old and new domains. Extensive experiments validate the superiority of OKR, outperforming the best exemplar-free method by significant margins of +5.6% and +6.5% mAP on the Pascal VOC and BDD100K series, respectively.

- **整体含义**：论文旨在探索一种**无样本**、**无路由**的持续适应机制，使检测器能在增量域上实现稳定 vs 可塑的平衡。核心洞察是：主干微调虽提供足够可塑性，但共享参数空间中的梯度更新会破坏旧表征；而通过低秩解耦与正交约束，可将不同域的学习隔离到独立子空间，避免干扰并保持紧凑模型。

### 2. 方法论

OKR 建立在 **ViTDet** 检测器之上，冻结预训练主干并注入可学习的 **LoRA** 模块。整体框架包括三个关键组件：

#### 2.1 低秩子空间扩展（Low-Rank Subspace Expansion）
- 为每个新域增量添加一对低秩矩阵 $(A_t, B_t)$，构成该域的专用子空间，旧域的参数 $\{A_{1:t-1}, B_{1:t-1}\}$ 保持冻结。
- 利用 LoRA 的线性可加性，将所有分支融合进统一投影矩阵：
  $$W_t = W_0 + \sum_{i=1}^t B_i A_i$$
  一次前向传递即可实现多域整体决策，无需域选择或额外推理开销。
- 训练时仅更新新分支，避免直接覆写历史参数。

#### 2.2 基于梯度的正交刷新（Gradient-Based Orthogonal Refreshing, GOR）
- 目标：保证新任务的梯度方向正交于旧任务的特征子空间，减少知识干扰。
- **历史子空间近似**：使用旧融合权重 $W_{t-1}$ 在新域数据 $D_t$ 上提取特征矩阵 $R_t$，进行 SVD：
  $$R_t = f_\Phi(W_{t-1}, D_t)$$
  $$SVD(R_t) = U_t \Sigma_t V_t^\top$$
  选取前 $k$ 个左奇异向量构成历史梯度子空间 $M_t = \text{span}\{u^1_t, \dots, u^{k_t}\}$。
- **梯度投影**：训练新任务时，将当前梯度 $\nabla_w L_t$ 投影到 $M_t$ 上并减去，确保更新仅沿正交补方向进行：
  $$\nabla_w L_t \leftarrow \nabla_w L_t - (\nabla_w L_t) M_t M_t^\top$$
  该投影迫使模型学习与旧知识互补的新特征，实现无遗忘的持续适应。

#### 2.3 拓扑感知一致性（Topology-Aware Consistency, TAC）
- **动机**：各域独立子空间学习可能导致同一类别在不同域中的语义结构分裂，损害跨域泛化。
- **做法**：以第一个域的类别原型 $\mu_1$ 作为全局参考拓扑。对新域 $t$，计算原型 $\mu_t$ 并强制其与对应的 $\mu_1$ 对齐：
  $$L_{\text{tac}} = \frac{1}{N} \sum_{i,j} w_{ij}^t \big(1 - \cos(\mu_i^t, \mu_j^1)\big)$$
  其中 $w_{ij}^t$ 为匹配置信度，促进同类特征的聚类与旧域对齐。
- **总损失**：$L_{\text{final}} = L_{\text{det}} + L_{\text{tac}}$（新域训练时加入 TAC 项，首域仅用检测损失）。

### 6. 主要结论与发现

- OKR 在所有基准上均显著优于无样本方法：Pascal VOC 最终会话 mAP 65.2%（+5.6%），BDD100K 系列 58.2%（+6.5%），VOC-Corruption 61.0%（+10.3%），甚至超越部分使用样本回放的方法。

### 7. 优点

- **方法简洁有效**：利用 LoRA 线性可加性实现零开销融合，避免复杂路由或额外存储。
- **理论驱动**：基于梯度‑输入空间关系的正交投影策略，从原理上减少干扰，实现了近似无遗忘的增量学习。
- **全面的实证评估**：在多个风格、场景、腐蚀类型的数据集上对比大量基线，消融和分析详尽，结论可靠。
- **实用性强**：计算开销低，显存节省，推理速度快，可轻松嵌入现有 ViT 检测器。
- **无样本约束**：符合隐私敏感场景需求，无需储存旧域样本。

### 8. 不足与局限

- **仅适用于固定标签空间**：OKR 假定类别集合在域间不变，无法直接处理类别增量问题，需在标签空间变化时扩展。
- **基域原型依赖**：TAC 以第一域原型为锚点，若首域质量差或类别间语义相似，可能引入偏差，且未探讨多锚点或自适应锚点策略。
- **实验未覆盖极端域差距**：虽包含风格和腐蚀变化，但未在更极端的跨模态（如 RGB → 红外）或强烈视角变化下验证。
