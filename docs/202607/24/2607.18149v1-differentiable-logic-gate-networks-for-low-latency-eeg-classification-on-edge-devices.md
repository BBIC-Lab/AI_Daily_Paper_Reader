## 摘要
边缘设备上的实时脑电图分类受到传统神经网络浮点运算的瓶颈限制。我们研究了可微逻辑门网络（Diff-Logic）作为一种硬件原生的替代方案，它将模型编译为纯布尔电路，可通过CPU位运算执行。通过在四个脑电图数据集上进行严格的等参数实验，涵盖两个分类任务（二分类痴呆检测和三分类情绪识别），我们在四个复杂度级别（5万至50万个参数）下，将Diff-Logic与容量匹配的多层感知器（MLP）和二值化神经网络（BNN）基线进行了比较。在痴呆筛查上，Diff-Logic取得了80.2%的宏平均F1分数，比MLP基线高出6.8%。在情绪识别上，MLP保持了适度的性能优势，但在部署于功耗受限（7瓦）的Nvidia Jetson Orin Nano CPU（单核）时，其延迟高出2.3倍，模型体积增大14倍。关键的是，Diff-Logic的推理时间在模型规模增加10倍时几乎保持不变，在最大复杂度级别上相对于MLP实现了最高2.9倍的加速。我们的结果确立了基于逻辑的神经架构作为资源受限的脑机接口的实用范式，在原生满足便携式边缘部署的延迟和内存约束的同时，取得了有竞争力或更优越的性能。代码已在GitHub上开源：https://github.com/Shyamal-Dharia/eeg-difflogic

## Abstract
Real-time EEG classification on edge devices is bottlenecked by the floating-point arithmetic of conventional neural networks. We investigated Differentiable Logic Gate Networks (Diff-Logic) as a hardware-native alternative that compiles models into pure Boolean circuits executable via bitwise CPU operations. Through rigorous iso-parameter experiments across four EEG datasets spanning two classification tasks, binary dementia detection and 3-class emotion recognition, we compared Diff-Logic against matched-capacity Multi-Layer Perceptron (MLP) and Binarized Neural Network (BNN) baselines at four complexity tiers (50k-500k parameters). On dementia screening, Diff-Logic achieved 80.2% Macro F1, outperforming the MLP baseline by 6.8%. On emotion recognition, the MLP retained a moderate performance advantage but incurred a 2.3$\times$ higher latency and 14$\times$ larger model size when deployed on a power-constrained (7W) Nvidia Jetson Orin Nano CPU (Single-core). Critically, Diff-Logic inference time remained nearly constant across a 10$\times$ increase in model scale, achieving a peak speedup of 2.9$\times$ over MLPs at the largest complexity tier. Our results establish logic-based neural architectures as a practical paradigm for resource-constrained brain-computer interfaces, achieving competitive or superior performance while natively satisfying the latency and memory constraints of portable edge deployment. Code is available on GitHub: https://github.com/Shyamal-Dharia/eeg-difflogic

### 2. 论文提出的方法论

- **核心思想**：采用可微逻辑门网络（Diff-Logic），将其训练成由可学习布尔逻辑门构成的稀疏网络。训练后，每个逻辑门“硬化”为单一布尔操作（如与、或、异或），整个模型最终编译成一个无乘加运算、仅通过CPU位运算即可执行的纯布尔电路。
- **关键技术细节**：
    - **输入量化与编码**：连续EEG特征首先通过**温度计编码**转换为二进制向量。对于归一化特征 $x'_j$ 和预设的分辨率 $T=15$，其二进制表示为 $b^{(j)}_k = \mathbb{I}(x'_j \ge \tau_k)$，其中 $\tau_k$ 是均匀分布的阈值。这保留了信号的序数关系，并将输入维度扩大至原来的 $T$ 倍。
    - **软逻辑门与训练**：网络中的每个神经元连接两个二进制输入 $(x_1, x_2)$，并维护一个可学习的概率分布 $\alpha_n \in \mathbb{R}^{16}$，对应全部16种二元布尔函数 $f_i$。训练时，神经元输出为所有函数的概率加权和：
        $$y_{\text{soft}} = \sum_{i=1}^{16} \frac{\exp(\alpha_{n,i})}{\sum_j \exp(\alpha_{n,j})} \cdot f_i(x_1, x_2)$$
        该过程完全可微，允许通过梯度下降进行端到端训练。输出层的组求和运算符将最终层的比特分组求和，并通过温度缩放的Softmax函数生成类别预测。
    - **编译与推理**：训练收敛后，每个神经元永久选择概率最高的那个函数，网络硬化成一个静态的、稀疏的布尔电路。推理时，该电路被编译为底层机器码，直接调用算术逻辑单元进行位运算（如与、或、非、异或），无需任何浮点运算或累加寄存器。

- **性能竞争力**：在痴呆检测任务上，Diff-Logic显著优于MLP和BNN基线，最高实现了**80.2%的宏平均F1分数**，领先MLP近10个百分点。在情绪识别任务上，Diff-Logic在50万参数级别**取得了与MLP相当的性能（61.0% vs. 61.3%）**。

- **极致的硬件友好性**：在50万参数级别，Diff-Logic模型仅占用**140KB存储空间**，比MLP**缩小了14倍**，且延迟仅为**0.22毫秒**，完全满足实时便携式脑机接口的需求。
- **架构优势**：研究表明，直接学习逻辑门和连接关系的Diff-Logic，比仅做权重量化的BNN在硬件效率上具有根本性优势，其无累加器的纯粹组合逻辑电路是关键。

### 8. 不足与局限

- **对预提取特征的依赖**：当前方法直接作用于手工提取的功率谱密度和差分熵特征，而非原始脑电图波形，无法实现端到端学习，可能损失了原始信号中的潜在信息。

- **应用领域有限**：当前评估仅限于两种特定EEG应用（痴呆与情绪），其在更多样化的BCI范式（如运动想象、事件相关电位）上的通用性有待进一步验证。
- **缺乏可解释性**：虽然网络由逻辑门组成，但如何从编译后的电路中提取出人类可理解的、有临床意义的布尔决策规则，仍是一个未解决的挑战。
