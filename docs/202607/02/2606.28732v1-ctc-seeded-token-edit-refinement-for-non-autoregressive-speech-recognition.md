## 摘要
非自回归自动语音识别（ASR）支持并行解码，但许多基于精炼的方法从随机、完全掩蔽或固定长度的令牌序列开始，需要多次迭代来重建完整转录。我们将ASR解码形式化为对贪婪连接主义时间分类（CTC）假设的变长编辑精炼。声学条件化的编辑流解码器直接在收缩的CTC假设上运行，并行预测插入、删除和替换操作。编辑流解码器使用连续时间离散扩散损失与CTC模型联合训练。在推理过程中，我们发现仅两个编辑步骤就可显著降低词错误率（WER），而无分类器引导（CFG）通过使模型聚焦于音频特征进一步提升了识别质量。我们还使用CTC置信度约束编辑提议以提高准确性。最后，消融研究验证了我们的设计选择，而解码器预训练和预训练编码器的集成则带来了显著的额外性能增益。

## Abstract
Non-autoregressive automatic speech recognition (ASR) enables parallel decoding, but many refinement-based methods begin from random, fully masked, or fixed-length token sequences, requiring multiple iterations to reconstruct the complete transcript. We instead formulate ASR decoding as a variable-length edit refinement of a greedy connectionist temporal classification (CTC) hypothesis. An acoustic-conditioned Edit Flow decoder operates directly on the collapsed CTC hypothesis, predicting insertion, deletion, and substitution operations in parallel. The Edit Flow decoder is jointly trained with a CTC model using a continuous-time discrete diffusion loss. During inference, we find that just two edit steps yield substantial Word Error Rate (WER) reductions, and classifier-free guidance (CFG) further enhances recognition quality by focusing the model on audio features. We also constrain edit proposals using CTC confidence to improve accuracy. Finally, ablation studies validate our design choices, while decoder pretraining and pretrained encoder integration yield significant additional performance gains.

### 2. 方法论

- **核心思想**：以贪婪 CTC 解码产生的初步转录（已去除空白符）为起始序列 $x_0$，利用声学条件化的 **编辑流解码器（Edit Flow Decoder）**，通过连续时间离散扩散过程预测编辑操作，逐步将 $x_0$ 精炼为目标转录 $x_1$。
- **关键技术细节**：
    - **首遍 CTC**：声学编码器输出帧级后验，通过贪婪对齐和收缩得到初始假设 $\hat{Y}^{(0)}$。
    - **可变长编辑路径**：在 Levenshtein 对齐的辅助空间中定义包含间隙符号 $\epsilon$ 的对齐序列 $z_0, z_1$。用单调噪声调度器 $\kappa(t)$ 混合中间状态 $z_t$，然后去除 $\epsilon$ 获得当前可见序列 $x_t$。
    - **声学条件编辑场**：解码器（双向 Transformer）以 $x_t$、时间 $t$ 和声学记忆 $H$ 为条件，对每个位置预测：
        - 插入强度 $\lambda^{\text{ins}}_i$ 和 token 分布 $q^{\text{ins}}_i(v)$ （对所有位置 $i$）；
        - 删除强度 $\lambda^{\text{del}}_i$、替换强度 $\lambda^{\text{sub}}_i$ 和 token 分布 $q^{\text{sub}}_i(v)$ （仅对非 $\langle\text{bos}\rangle$ 位置）。
    - **扩散目标**：使用编辑流扩散损失 $L_{EF}$，将目标编辑索引集 $M_t$ 中的编辑率与解码器预测的 edit intensity 对齐。联合训练总损失为 $L = \alpha L_{CTC} + (1-\alpha)L_{EF}$，其中 $\alpha=0.3$。
    - **编辑感知文本预训练**：先用纯文本数据，对干净文本独立应用删除、替换、插入噪声生成源序列，用相同编辑流目标预训练解码器（此时声学记忆 $H=0$），以增强纠正 ASR 错误的能力。
- **推理策略**：
    - **多步迭代精炼**：采用 $K=2$ 步，每步用 Poisson tau-leaping 近似计算插入和删除/替换的事件概率，确定性解码接受概率高于阈值（0.1）的编辑。
    - **音频无分类器引导（CFG）**：训练时以 0.1 概率丢弃声学条件，推理时用引导尺度 $w$ 融合有/无音频条件的预测，使模型更聚焦声学证据。
    - **CTC 置信度引导**：基于 CTC 对齐计算每个 token 的置信度 $c_i$ 和边界置信度 $b_j$。编辑提案仅在事件概率高于阈值且对应置信度低于阈值（0.7）时才被接受。首步用贪婪对齐的置信度，后续步骤用强制对齐的置信度。

### 6. 主要结论与发现

- **编辑精炼高效**：只需两次并行编辑迭代即可实现远优于 CTC 基线的 WER。

- **最终性能**：
    - ESPNet Large 编码器：test-clean/test-other 的 WER 从 3.5/7.9 降至 2.6/5.8 (相对改善 25.7%/26.6%)。
    - Whisper Medium 编码器：WER 从 2.6/6.1 降至 **2.0/4.7** (相对改善 23.1%/23.0%)。
    - 以较少参数量和训练数据，取得了与依赖大规模预训练或强监督初始化的先进方法可比的结果。

### 7. 优点

- **新颖的范式**：将 ASR 解码转换为可变长令牌级编辑，完全避免了生成空白符或维持固定帧级对齐的冗余。
- **极度高效的推理**：仅需两步并行编辑即可收敛，极大降低了迭代次数。
