## 摘要
有效的脑疾病诊断需要脑连接模式与高层语义知识的协同。然而，现有方法大多将大语言模型（LLM）的语义视为辅助特征或监督，限制了其在决策中的直接作用，并制约了分类的稳定性和鲁棒性。为此，我们提出了一种语义对齐的脑网络框架，将LLM衍生的语义主动整合到预测过程中。具体来说，首先通过全局自注意力融入ROI级别的语义，以丰富节点表示并提供全脑上下文。然后构建多尺度超图，显式建模功能子网络和多ROI交互，解决传统图神经网络的局部性限制，捕获高阶依赖关系。最后，决策级语义对齐机制选择性地将患者特定的文本嵌入注入图表示，使语义能够直接指导预测，而不会扰动底层网络结构。在公开脑网络数据集ABIDE和ADHD-200上的实验表明，该方法达到了最先进的性能，增强了稳定性和可解释性，尤其在小样本环境中表现突出。

## Abstract
Effective brain disease diagnosis requires the synergy of brain connectivity patterns and high-level semantic knowledge. Existing methods, however, largely treat semantics from large language models (LLMs) as auxiliary features or supervision, limiting their direct role in decision-making and constraining classification stability and robustness. To overcome this, we propose a semantic-aligned brain network framework that actively integrates LLM-derived semantics into the prediction process. Specifically, ROI-level semantics are first incorporated via global self-attention to enrich node representations and provide whole-brain context. Multi-scale hypergraphs are then constructed to explicitly model functional subnetworks and multi-ROI interactions, addressing the locality limitations of traditional GNNs and capturing high-order dependencies. Finally, a decision-level semantic alignment mechanism selectively injects patient-specific textual embeddings into graph representations, enabling semantics to directly guide predictions without perturbing the underlying network structure. Experiments on public brain network datasets ABIDE and ADHD-200 demonstrate state-of-the-art performance, enhanced stability, and improved interpretability, particularly in small-sample settings.

### 2. 论文提出的方法论
**核心思想**：分层注入语义先验，从节点表示到图级抽象再到决策输出，逐步让语义与脑网络结构对齐。具体分为三个模块：

- **Module I：多尺度节点级脑网络编码**
  - **节点特征初始化**：基于 AAL 脑图谱划分 $N$ 个 ROI，计算皮尔逊相关得到初始特征矩阵 $X \in \mathbb{R}^{N \times N}$。
  - **ROI 语义嵌入注入**：利用 LLM 为每个 ROI 生成解剖位置与功能描述，通过冻结的文本编码器得到语义嵌入 $E_{\text{ROI}}$，经线性投影后与 $X$ 相加，再通过 Brain Network Transformer 的全局自注意力编码，得到语义增强的节点特征 $X_0 = \text{BNT}(X + E_{\text{ROI}} W_p)$。
  - **多尺度超图建模**：基于 $X_0$ 构建不同尺度 $k$ 的超图，超边连接语义功能相似的 ROI 组。每个尺度通过两层超图神经网络传播特征，再用共享参数的多头自注意力模块统一交互模式，生成多尺度节点表示 $\{Z^{(k)}\}$。

- **Module II：图级表示抽象**
  - **尺度内池化**：对每个尺度的节点特征 $Z^{(k)}$ 做全局平均池化，得到尺度特异的图描述向量 $g^{(k)} = \frac{1}{N}\sum_{i=1}^N Z_i^{(k)}$。
  - **自适应多尺度融合**：拼接所有 $g^{(k)}$，通过一个可学习的门控网络输出 softmax 权重 $\alpha$，加权求和得到统一图表示 $g = \sum_{k=1}^K \alpha_k g^{(k)}$。

- **Module III：语义对齐与决策级增强**
  - **患者特异性语义注入**：引入一个可训练的 Graph‑CLS token $q_{\text{CLS}}$，与图表示 $g$ 相加形成条件查询 $q = q_{\text{CLS}} + g$；将患者元数据（人口学、临床文本）经 LLM 编码为嵌入序列 $E_{\text{Patient}}$，使用 $q$ 对其进行交叉注意力，得到融合表示 $h_{\text{fused}}$。
  - **门控残差注入**：通过 $g' = g + \sigma(W_g h_{\text{fused}}) \odot h_{\text{fused}}$ 自适应控制语义信息的贡献，防止噪声干扰。
  - **双重监督**：交叉熵损失 $\mathcal{L}_{\text{CE}}$ 确保分类准确性，对比损失 $\mathcal{L}_{\text{CL}}$ 将 $g'$ 与 LLM 生成的疾病类嵌入对齐，总损失为 $\mathcal{L}_{\text{total}} = \mathcal{L}_{\text{CE}} + \lambda \mathcal{L}_{\text{CL}}$。
