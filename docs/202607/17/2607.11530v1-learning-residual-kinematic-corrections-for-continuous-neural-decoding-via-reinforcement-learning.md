## 摘要
基于非侵入性脑电图（EEG）的脑机接口（BCI）在解码连续三维（3D）运动想象（MI）时，由于信号变异性和残余解码误差，仍具挑战性。卷积神经网络-长短期记忆（CNN–LSTM）等深度学习架构可捕捉时空动态以实现连续运动学解码，但预测轨迹中仍存在系统性残差误差。我们提出一种两阶段解码框架，应用强化学习（RL）对CNN–LSTM解码器（CNN–LSTM–RL）的输出进行残差运动学校正。RL智能体在离线状态下训练，不直接接收EEG输入，而是基于预测的运动学轨迹优化相对于目标轨迹的运动精度。解码性能通过沿x、y和z轴的皮尔逊相关系数（r）和均方根误差（RMSE）量化。与单独使用CNN–LSTM相比，CNN–LSTM–RL在二维条件下将平均相关系数从0.5076提升至0.7181（p = 0.0005），在虚拟现实中从0.6420提升至0.7780（p = 0.0059），相对增益分别为41.5%和21.2%。相应地，RMSE在二维条件下从0.0890降至0.0532（p < 0.0001），在虚拟现实中从0.0714降至0.0441（p < 0.0001），相对降幅分别为40.2%和38.2%。这些发现表明，该可扩展框架通过离线残差RL校正运动学误差，无需额外神经数据，从而增强了3D BCI MI解码能力，推动了神经康复、假肢和虚拟交互的发展。

## Abstract
Decoding continuous three-dimensional (3D) motor imagery (MI) using non-invasive electroencephalography (EEG)-based brain--computer interfaces (BCIs) remains challenging due to signal variability and residual decoding errors. Deep learning architectures such as convolutional neural network--long short-term memory (CNN--LSTM) models can capture spatial and temporal dynamics for continuous kinematic decoding; however, systematic residual errors persist in predicted trajectories. We propose a two-stage decoding framework that applies reinforcement learning (RL) to perform residual kinematic correction on the outputs of a CNN--LSTM decoder (CNN--LSTM--RL). The RL agent is trained offline without direct EEG input and instead operates on predicted kinematic trajectories to optimize movement accuracy relative to target trajectories. Decoding performance was quantified using Pearson correlation coefficients ($r$) and Root Mean Square Errors (RMSE) along the $x, y$, and $z$ axes. Compared to CNN--LSTM applied alone, CNN--LSTM--RL improved the mean correlation from $0.5076$ to $0.7181$ ($p = 0.0005$) in 2D and from $0.6420$ to $0.7780$ ($p = 0.0059$) in VR, with relative gains of $41.5\%$ and $21.2\%$, respectively. Correspondingly, RMSE was reduced from $0.0890$ to $0.0532$ (2D, $p < 0.0001$) and from $0.0714$ to $0.0441$ (VR, $p < 0.0001$), representing relative reductions of $40.2\%$ and $38.2\%$. These findings demonstrate that this scalable framework enhances 3D BCI MI decoding by correcting kinematic errors via offline residual RL without extra neural data, advancing neurorehabilitation, prosthetics, and virtual interaction.

## 2. 方法论  
- **两阶段框架**：  
  - 第一阶段：基于事件相关谱扰动（ERSP）的 CNN–LSTM 解码器，从 17 通道 EEG 中解码三维速度 $v^{\text{BCI}}_t$。  
  - 第二阶段：离线 SAC 智能体接收观测 $o_t = [v^{\text{BCI}}_t, v_{t-1}, t]$（上次校正速度与归一化时间），输出残差速度 $v^{\text{res}}_t$，得到最终速度 $v_t = v^{\text{BCI}}_t + v^{\text{res}}_t$。  
- **RL 问题建模**：离散时间 POMDP。状态 $s_t$ 包含真实 EEG、目标速度等隐藏信息；动作 $a_t$ 为残差速度；奖励函数 $r_t = -d(v_t, v^*_t)$，$d$ 为欧氏距离。  
- **算法流程**：使用 Soft Actor-Critic，优化带熵的目标 $J(\pi)=\mathbb{E}_{(s_t,a_t)\sim\pi}\big[r(s_t,a_t)+\alpha\mathcal{H}(\pi(\cdot|s_t))\big]$。策略通过重参数化 $a_t=\text{scale}\cdot\tanh(\mu(s_t)+\sigma(s_t)\odot\epsilon_t)$ 输出有界动作，并修正对数概率。  
- **多目标优化**：用 Optuna 最大化皮尔逊相关系数以选择超参数，RL 自身最小化轨迹欧氏距离，实现形态与数值精度的协同优化。  
- 关键点：RL 智能体离线训练，不接触 EEG 原始信号，仅依赖解码器输出，实现校正与神经噪声的解耦。

## 3. 实验设计  
- **数据集**：10 名右利手被试，每人完成 10 个 session，交替 2D 屏幕反馈与沉浸式 VR 反馈。任务为想象到达四个 3D 目标，共 25,600 条试验。  
- **运动学数据**：60 Hz 采集的腕部 Vive 追踪器提供真实三维坐标（x，y，z），派生速度作为目标。  
- **对比策略**：  
  - FDG：固定解码器，Session 1 训练，后续不变。  
  - SAT：前一 session 训练，下一 session 测试（周期性重校准）。  
  - WSR：同一 session 内训练并测试（能力上界）。  
  - FRL：固定 RL 智能体，Session 1 训练，全程应用残差校正。  
- **评估指标**：逐试次平均的皮尔逊相关系数 $r$ 与均方根误差 RMSE。  
- **统计检验**：对 2D 和 VR 分别进行重复测量 ANOVA 或 Friedman 检验，事后成对比较采用配对 t 检验或 Wilcoxon 符号秩检验（视分布而定），并做 Holm-Bonferroni 校正。

## 4. 资源与算力

## 5. 实验数量与充分性  
- 主体实验涵盖 10 名被试×两种反馈环境（2D 与 VR）×4 种策略对比，总计约 80 组条件（若考虑每个 session 的统计）。

## 6. 主要结论与发现  
- FRL 在所有测试 session 中均获得最高相关系数和最低 RMSE。  
- 与 FDG 相比，2D 下 $r$ 提升 41.5%，RMSE 降低 40.2%；VR 下 $r$ 提升 21.2%，RMSE 降低 38.2%，均达高度统计显著（$p<0.001$）。  
- 大幅超越周期性重校准策略 SAT，且在多数情况下超越能力上界 WSR。  
- Y 轴和 Z 轴的改善尤为突出，$r$ 最高超 0.98。  
- 框架展现出优良的跨会话稳定性，对信号漂移具有鲁棒性。

## 7. 优点  
- **解耦设计**：离线 RL 智能体不直接暴露于高噪声 EEG，校正过程与基础解码器完全分离，提升鲁棒性。  
- **高效优化**：多目标（相关性+距离）超参数搜索与奖励协同，在不增加在线训练负担下显著增强性能。  
- **统计严谨**：对不同分布采用适配的假设检验，多重比较校正，效果量完整。  
- **实用性强**：可扩展至多种反馈模态，且与在线流程兼容（固定模型即可应用），达到已知 SOTA 提升。

## 8. 不足与局限  
- **初始解码器依赖**：残差校正效果受限于基线 CNN–LSTM 的质量；若基础解码器极差，RL 亦难挽回。  
- **时间信息敏感**：智能体观测中高度依赖归一化时间步 $t$，去掉后性能骤降，不利于时间非同步的实时 BC。  
- **任务简化**：仅包含四个固定目标的有序到达任务，未覆盖更复杂的连续轨迹或更多目标，泛化性待验证。  
- **离线评估**：全部分析基于离线数据，未进行实时在线实验以验证闭环下的用户适应性与交互体验。  
- **超参敏感性**：动作缩放因子等对性能影响较大，实际部署需谨慎调节。
