## 摘要
多变量时间序列的异常检测在大规模分布式系统中仍是一项关键挑战，这些系统中相关实体可能在异常发生前表现出可迁移的先兆行为。现有方法通常独立处理每个数据流，因此本质上仍是被动的。为解决这一局限，我们引入分布式分层时序记忆（D-HTM），这是一个通过共享联想记忆（SAM）实现跨实体抢先预警的神经形态框架。D-HTM 结合了空间池化器（SP）将观测值映射到一个共同的稀疏分布式表征（SDR）空间、在线学习实体特定动态的时序记忆（TM）模块，以及存储重复出现的异常前征兆的共享联想记忆。通过跨相关实体重用先兆知识，D-HTM 能够在局部异常发生前发出警告，同时保留 HTM 的在线学习能力。我们在服务器机器数据集（SMD）、土壤水分主动被动（SMAP）数据集、火星科学实验室（MSL）数据集以及一个旨在隔离先兆迁移的合成级联基准上评估了 D-HTM。实验结果表明，有效的跨实体警告传播，同时保持有竞争力的被动异常检测性能。在真实世界数据集上，D-HTM 在异常发生前平均提供了 8.1 个样本的预警提前时间。这些发现表明，可迁移的先兆结构可以在共享 SDR 空间中涌现，并被重用于生成抢先预警，将 HTM 从孤立的被动检测扩展到分布式预测推理。

## Abstract
Anomaly detection in multivariate time series remains a critical challenge in large-scale distributed systems, where related entities may exhibit transferable precursor behavior prior to anomaly onset. Existing methods typically operate independently on each data stream and therefore remain fundamentally reactive. To address this limitation, we introduce Distributed Hierarchical Temporal Memory (D-HTM), a neuromorphic framework that enables cross-entity preemptive warning through a Shared Associative Memory (SAM).   D-HTM combines a Spatial Pooler (SP) that projects observations into a common Sparse Distributed Representation (SDR) space, Temporal Memory (TM) modules that learn entity-specific dynamics online, and a Shared Associative Memory that stores recurring pre-anomaly signatures. By reusing precursor knowledge across related entities, D-HTM can issue warnings prior to local anomaly onset while preserving HTM's online learning capabilities.   We evaluate D-HTM on the Server Machine Dataset (SMD), the Soil Moisture Active Passive (SMAP) dataset, the Mars Science Laboratory (MSL) dataset, and a synthetic cascade benchmark designed to isolate precursor transfer. Experimental results demonstrate effective cross-entity warning propagation while maintaining competitive reactive anomaly detection performance. Across the real-world datasets, D-HTM provides an average warning lead time of 8.1 samples prior to anomaly onset.   These findings demonstrate that transferable precursor structure can emerge within a shared SDR space and be reused for preemptive warning generation, extending HTM beyond isolated reactive detection toward distributed predictive reasoning.

## 1. 论文的核心问题与整体含义

- **研究动机**：大规模分布式系统（如服务器集群、航天器遥测）中，多变量时间序列异常检测至关重要。异常往往在相关实体间传播，并在异常发生前表现出可迁移的“先兆行为”（precursor behavior）。
- **现有方法局限**：当前方法（包括 HTM、VAE、LSTM 等）通常独立处理每个数据流，只能在异常于自身观测到后才响应，本质上是被动的（reactive）；即使多智能体或联邦学习方法，推理阶段也不进行跨实体知识交换。
- **整体意图**：提出分布式分层时序记忆（D-HTM）框架，利用**共享联想记忆（SAM）** 将 HTM 从孤立的反应式检测扩展到**跨实体的抢先预警**，在异常真正发生之前利用其他实体学到的先兆知识发出警告。

## 2. 论文提出的方法论

### 2.1 核心思想
- 为每个实体保留独立的**时序记忆（TM）** 以在线学习实体特有的时序动态，同时所有实体共享一个**空间池化器（SP）** 和一个**共享联想记忆（SAM）**。
- SP 将各实体经过 z-score 归一化的输入映射到公共的**稀疏分布式表征（SDR）** 空间，SAM 存储异常发生前 $L$ 步的 SP 激活模式（“前兆记忆”）。
- 在线推理时，实时构造查询窗口与 SAM 中的记忆进行匹配，若成功则发出预警，实现**推理时的跨实体知识传递**。

### 2.2 关键技术细节

- **特征归一化**：每个实体 $i$ 的特征 $j$ 独立进行 z-score 归一化，消除操作点偏差，使不同实体下的相似行为更具可比性：
  $$z_{i,j}(t) = \frac{x_{i,j}(t) - \mu_{i,j}}{\sigma_{i,j}}$$
- **共享空间池化器**：使用代表历史数据离线训练一次后冻结，保证相似输入产生重叠的 SDR，从而支撑跨实体的相似性比较。
- **局部时序记忆**：每个实体维持独立的 TM，在线学习时序模式并输出预测误差作为异常分数。
- **共享联想记忆（SAM）与警告传播**：
  - 记忆条目 $M_j = (W_j, n_j, t_j)$，包含前兆窗口 $W_j$（$L$ 个 SDR）、复发计数 $n_j$ 和最近访问时间。
  - 当 HTM 检测到异常起始时，提取前 $L$ 步 SP 激活并存入 SAM（若与现有记忆相似则增加计数，否则新条目）。
  - 推理时，每个实体构建查询窗口 $Q_t = \{s_{t-L}, \dots, s_{t-1}\}$，对每个记忆进行**位置匹配**：若 $|q_p \cap w_p^j| \ge \tau$，则当前位置匹配。
  - 当匹配位置数 $\ge K$（K-of-L 规则），SAM 发出抢先预警：
    $$\text{warning}(t) = \max_j \left( \sum_{p=0}^{L-1} \mathbf{1}[|q_p \cap w_p^j| \ge \tau] \right) \ge K$$
- **在线算法流程**（算法 1）：先在 SAM 中查询，后若发生异常则更新 SAM，记忆淘汰基于复发计数和最近访问时间。

### 2.3 方法特色
- **无全局同步，无模型重训练**；每个实体独立学习本地时序，仅通过共享 SP 和 SAM 交换“先兆记忆”，保持在线学习能力。
- **共享 SP** 替代独立 SP 是跨实体表征对齐的关键，消融实验证明共享 SP 的预警 F1 明显高于原始特征、编码器 SDR 或独立 SP。

## 6. 主要结论与发现

- D‑HTM 可在保持有竞争力反应式检测性能（如针对 SMD 的 F1 0.815 vs OmniAnomaly 0.879）的同时，实现有效的**跨实体抢先预警**。
- 在真实数据集上平均预警提前量为 **8.1 个样本**，MSL 上最高达 7.21 样本，合成级联达 11.36 样本。
- **共享空间池化器是跨实体预警的关键**：其表征对齐能力远优于独立 SP 或原始特征，共享 SP 将宏观平均 F1 从原始特征的 0.29 提升到 0.82。
- 前兆记忆检索中 **$K/L = 0.6$ 为较好的平衡点**，前兆窗口长度 $L$ 因数据集而异。
- Shuffled control 中预警 F1 从 0.82 骤降至 0.05，证明 SAM 的性能来源于真实先兆结构而非偶然重叠。

## 7. 优点

- **创新性**：首次在 HTM 框架中引入**推理时跨实体知识传递**，通过共享联想记忆存储和重用前兆模式，无需参数共享或模型更新。
- **分布式、在线且异步**：各实体独立学习本地时序，只依赖统一的 SDR 空间和 SAM 进行稀疏激活匹配，符合生物启发式并行。
- **表征对齐设计**：利用 z‑score 归一化消除操作点差异，配合共享 SP 实现不同实体间的语义相似性比较，消融实验验证了各组件贡献。
- **实验扎实**：覆盖多个真实基准与合成场景，包含反应式检测、警告传播、表征消融、参数敏感性、控制实验等多维度评估。
- **实用前景**：早期预警能力可应用于数据中心、航天器等场景，为分布式预测推理提供新思路。

## 8. 不足与局限

- **固定前兆窗口**：$L$ 为数据集依赖的超参数，缺乏自适应机制，可能无法捕捉不同时间尺度的前兆。
- **SP 冻结**：共享空间池化器离线训练后冻结，无法在部署后适应概念漂移或新实体，限制了长期部署的适应性。
- **匹配机制简单**：采用固定的重叠阈值 $\tau$ 和 K‑of‑L 匹配，无概率置信度估计或因果推理，可能产生误报或漏报。
- **缺乏与更多最新方法的直接对比**：虽然与 OmniAnomaly 比较了反应式检测，但未与近期其他联邦学习或分布式早期预警方法直接对比预警性能（仅概念比较）。
- **数据局限**：SMD 中 3 台机器因数值退化被排除，可能影响代表性和结果的普遍性；合成基准虽然理想，但不完全反映真实复杂性。
- **计算复杂度与实时性**：未讨论跨实体检索的内存/时间开销以及大规模部署的扩展性问题（尽管容量足够本实验，未触发淘汰策略）。
- **记忆管理策略**：仅基于复发计数、最近访问时间淘汰，对非平稳环境或长周期异常可能不够鲁棒。
