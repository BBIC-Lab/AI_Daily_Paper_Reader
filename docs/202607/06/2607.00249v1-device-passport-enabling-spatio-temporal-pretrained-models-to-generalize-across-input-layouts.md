## 摘要
新的设备布局由于缺乏每个特定布局的大规模数据集，构成了一项具有挑战性的建模问题。生物信号基础模型如果能有效地泛化到新布局，则提供了一种合理的解决方案。为了改善跨布局迁移，我们研究了当预训练布局与下游解码布局有显著差异时，不同的通道嵌入技术如何表现。我们提出了设备护照，一种新的通道嵌入技术，它学习专家模型和混合模型，这些模型将每个通道的功能活动和元数据作为输入。这与以往的嵌入方法形成对比，以往的方法通常仅使用功能信息或仅使用元数据来查找学习到的或固定的位置嵌入。在受控子集迁移实验和向耳脑电图（ear-EEG）的切实迁移中，设备护照在总体上具有竞争力，并在促使此项工作的布局迁移场景中，相较于最强的学习型基线有所改进。这些结果表明，在将大规模预训练生物信号模型重用于新设备时，通道嵌入设计是一个关键考虑因素。

## Abstract
New device layouts pose a challenging modeling problem due to the lack of large datasets for each specific layout. Biosignal foundation models offer a plausible solution if they are able to generalize to new layouts effectively. To improve cross-layout transfer, we study how different channel embedding techniques behave when pretraining layouts differ substantially from the downstream decoding layout. We propose Device Passport, a new channel embedding technique that learns experts and mixture models that take each channel's functional activity and metadata as input. This contrasts with prior embedding methods, which typically use only functional information or only metadata to look up learned or fixed positional embeddings. Across controlled subset-transfer experiments and realistic transfer to ear-EEG, Device Passport is competitive overall and improves over the strongest learned baseline in the layout-transfer regimes that motivate this work. These results suggest that channel embedding design is a key consideration when reusing large-scale pretrained biosignal models on new devices.

- **研究背景**：在脑电图（EEG）、肌电图（EMG）等生物信号领域，大规模预训练基础模型在下游任务中表现优异，但通常依赖预训练与下游使用相同的传感器布局（通道位置）。当新设备采用不同布局时，往往缺乏大规模标注数据，导致模型难以复用。
- **核心问题**：通道嵌入是 Transformer 感知空间信息的关键组件。现有的通道嵌入方法（如基于通道 ID 查找、绝对位置编码、仅依赖功能活动的条件编码）在布局发生显著变化、且下游数据稀缺时，其复用性严重受限。
- **整体含义**：论文提出“设备护照”（Device Passport），一种新的通道嵌入技术，旨在使时空预训练模型能够跨不同设备布局泛化，重点解决在仅用少量下游数据的情况下，如何有效迁移预训练的空间知识。

### 2. 方法论
- **核心思想**：不单独使用通道功能活动或元数据，而是将两者结合，通过一个**混合专家模型**来生成通道嵌入。预训练时学习一组专家嵌入和一个混合规则；下游迁移时冻结专家，仅微调混合规则，使模型能快速适应新布局。
- **具体设计（Device Passport）**：
  - 对于每个通道 $c$，首先将其非掩码补丁（patch）嵌入池化成一个活动摘要向量 $z_c$。
  - 将活动摘要 $z_c$ 与通道元数据（本文使用电极的XYZ坐标）$m_c$ 拼接，作为混合模型的输入。
  - **MLP 专家变体**：通过一个多层感知机计算混合权重 $w_c = \mathrm{softmax}(\mathrm{MLP}([z_c, m_c]))$，通道嵌入为所有专家嵌入的加权和：$e_c = \sum_k w_{c,k} E_k$。
  - **交叉注意力专家变体**：将 $[z_c, m_c]$ 作为查询（query），对所有专家对应的键（key）和值（value）进行注意力聚合，得到通道嵌入。
  - 专家数量为 10（固定），在预训练时与混合模块联合学习。
- **对比基线**：
  - 绝对位置编码（APE，正弦函数按通道序号索引）
  - 通道ID嵌入（可学习的通道名查找表）
  - XYZ嵌入（按电极坐标索引的正弦编码）
  - 非对称条件位置编码（ACPE，CBraMod 中使用的方法，仅从功能补丁嵌入通过二维卷积生成位置编码，无元数据）

### 6. 主要结论与发现
- **通道嵌入是跨布局迁移的关键瓶颈**。当预训练与下游布局不匹配时，所有方法的性能均下降。
- **Device Passport 在不匹配布局场景中总体最优**。在控制实验的 Temporal‑to‑Standard 和 Frontal‑to‑Standard 迁移，以及面向耳‑EEG 的真实迁移中，MLP 和 Cross‑Attention 专家变体在大多数受试者上优于最强学习基线 ACPE（7/9 受试者提升）。
- **ACPE 在极低数据量（<50 受试者）下仍有竞争力**，因其参数量少，易于快速适应；Device Passport 在拥有一定下游数据（约 50‑150 受试者）时开始显现优势。

### 7. 优点
- **提出了一种元数据与功能活动联合驱动的通道嵌入范式**，通过“预训练专家+微调混合”的方式实现空间知识的轻量迁移，无需重新学习整个通道表示。
- **实验设计系统且严谨**，同时考虑受控布局变化和真实设备迁移，覆盖低数据场景，并对多种嵌入方案进行了公平对比。

### 8. 不足与局限
- **对噪声/低质量通道鲁棒性不足**：当通道信号不可靠时，活动摘要失真，导致专家选择错误，性能下降。文中未提出显式的去噪或不确定性处理机制。

- **极低数据下的优势有限**：在只有极少数微调样本时，ACPE 可能更优，Device Passport 并非在所有样本规模下均占优。
